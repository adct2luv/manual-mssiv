# ส่วนประกอบ

## ตัวล็อกด้านนอก

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **เครื่องสแกนลายนิ้วมือ** | แตะนิ้วมือที่ลงทะเบียนเพื่อปลดล็อก |
| 2 | **ปุ่มเปิดฝาครอบลายนิ้วมือ** | กดเพื่อเปิดฝาครอบป้องกันเครื่องสแกนลายนิ้วมือ |
| 3 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |
| 4 | **แถบเซ็นเซอร์ทำงาน** | สัมผัสเพื่อเปิดใช้งานหน้าจอหรือระบบอ่านการ์ด |
| 5 | **Number pad** |  |
| 6 | **Battery lamp** |  |
| 7 | **ช่องเสียบกุญแจกลไก** | ไขกุญแจกลไกสำรองกรณีฉุกเฉิน |
| 8 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |
| 9 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 10 | **ไฟ LED แสดงสถานะ** | แสดงสถานะการทำงานของตัวล็อก (สแตนด์บาย / ล็อกแล้ว / ข้อผิดพลาด) |
| 11 | **ที่จับประตูด้านนอก** | ดึงหรือหมุนลงเพื่อเปิดประตูหลังปลดล็อกสำเร็จ |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |
| 2 | **ช่องใส่โมดูลรับสัญญาณ (อุปกรณ์เสริม)** | ช่องสำหรับติดตั้งโมดูลรีโมทคอนโทรลหรือสมาร์ทโฮม |
| 3 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 4 | **มอร์ทิส** |  |
| 5 | **สลักล็อก (Deadbolt)** | สลักล็อกหลักของประตูเมื่อประตูล็อก |
| 6 | **ลิ้นสลัก (Latch Bolt)** | ลิ้นสลักสปริงที่ยึดประตูกับวงกบขณะปิด |
| 7 | **ปุ่มล็อกที่จับด้านใน** | เปิด/ปิดระบบล็อกที่จับประตูด้านในเพื่อป้องกันเด็กหรือสัตว์เลี้ยงเปิดประตู |
| 8 | **แบตเตอรี่อัลคาไลน์ AA** | แหล่งพลังงานหลักสำหรับตัวล็อกประตูดิจิทัล |
| 9 | **ปุ่มปิด (ล็อกสองชั้น 3 วินาที)** | กดเพื่อปิดประตู หรือกดค้างไว้ 3 วินาทีเพื่อเปิดใช้งานระบบล็อกสองชั้น |
| 10 | **ลำโพง** | ส่งเสียงแนะนำการใช้งานหรือสัญญาณเตือนภัย |
| 11 | **ที่จับประตูด้านใน** | ดึงหรือหมุนเพื่อเปิดประตูจากภายในห้อง |

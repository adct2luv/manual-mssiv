# สัญญาณเตือนและความปลอดภัย

## สัญญาณเตือนการบุกรุก (การงัดแงะ)

เมื่อระบบตรวจพบความพยายามในการงัดแงะทำลายตัวเครื่องล็อกประตูไม่ว่าจะเป็นจากทางฝั่งด้านในหรือด้านนอกห้อง ตัวเครื่องจะส่งเสียงไซเรนแจ้งเตือนเหตุงัดแงะระดับสูงดังขึ้น

**การยกเลิก:** ถอดแบตเตอรี่ออก หรือป้อนรหัส PIN หรือทาบบัตรที่ลงทะเบียนเพื่อปิดสัญญาณเตือนภัย

## สัญญาณเตือนเซ็นเซอร์ตรวจจับไฟไหม้

เมื่อตรวจพบระดับความร้อนอุณหภูมิภายในตัวห้องสูงขึ้นถึงประมาณ (62°C±5°C) สัญญาณแจ้งเตือนความร้อนภัยจะทำงานและระบบจะปลดล็อกล็อกประตูอัตโนมัติทันทีเพื่อให้ปลอดภัย

:::caution[ข้อควรระวัง]
การที่ตัวเครื่องโดนแสงแดดส่องโดยตรงเป็นเวลานานอาจทำให้เซ็นเซอร์ตรวจจับความร้อนและเปิดเสียงสัญญาณเตือนภัยได้
:::

:::caution[ข้อควรระวัง]
แม้ว่าจะอยู่ในสถานะโหมดล็อกบังคับ (Force Lock) ประตูจะปลดล็อกโดยอัตโนมัติ และเมื่ออุณหภูมิกลับมาเป็นปกติ ตัวล็อกจะกลับสู่สถานะการทำงานปกติ
:::

## สัญญาณเตือนป้องกันการเดารหัสและการล็อก 1 นาที

การป้อนรหัส PIN หรือการทาบคีย์แท็กผิดติดต่อกัน 5 ครั้ง จะเปิดใช้งานสัญญาณเตือน และตัวล็อกจะไม่ทำงานเป็นเวลา 1 นาที

## สัญญาณเตือนสลักกลอนติดขัด

### ขั้นตอน

1. ระหว่างการเปิดและปิดประตู หากเซ็นเซอร์ตรวจจับไม่ทำงาน จะมีเสียงเตือนแสดงข้อผิดพลาดดังขึ้น
2. ระบบจะดำเนินการซ้ำสูงสุด 3 ครั้ง

## สัญญาณแจ้งเตือนการเปลี่ยนแบตเตอรี่

หากมีเสียงเตือนดังขึ้นในระหว่างการใช้งานปกติ แสดงว่าถึงเวลาที่ต้องเปลี่ยนแบตเตอรี่ โปรดเปลี่ยนแบตเตอรี่ใหม่ทั้งหมดในเวลานี้

:::caution[ข้อควรระวัง]
โปรดเปลี่ยนแบตเตอรี่ทั้งหมดพร้อมกันในครั้งเดียว
:::

:::caution[ข้อควรระวัง]
ห้ามใช้แบตเตอรี่เก่าและใหม่ร่วมกัน
:::

:::caution[ข้อควรระวัง]
ห้ามใช้แบตเตอรี่ต่างยี่ห้อร่วมกัน
:::

## แบตเตอรี่ฉุกเฉิน

เมื่อแบตเตอรี่หลักในตัวล็อกหมดลง ประตูสามารถปลดล็อกเปิดได้โดยใช้แบตเตอรี่สำรองฉุกเฉิน (ผู้ใช้งานต้องจัดหาซื้อแบตเตอรี่อัลคาไลน์ 9V แยกต่างหาก)

### ขั้นตอน

1. แตะขั่วแบตเตอรี่ 9V เข้ากับขั้วสัมผัสแบตเตอรี่ฉุกเฉิน
2. กดปุ่ม [*]
3. ป้อนรหัส PIN
4. กดปุ่ม [*] อีกครั้งเพื่อเปิดประตู

# ส่วนประกอบ

## ตัวล็อกด้านนอก

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **แผงคีย์แพดตัวเลข** | ป้อนรหัส PIN หรือรหัสตั้งค่าการทำงาน (1-9, *, 0, #) |
| 2 | **ไฟแจ้งเตือนสถานะแบตเตอรี่** | ไฟเตือนจะสว่างขึ้นเมื่อแบตเตอรี่อ่อน |
| 3 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 4 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |
| 5 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ปุ่มเปิด/ปิดประตู (Open/Close)** | กดปุ่มเพื่อล็อกหรือปลดล็อกประตูแบบสัมผัสเดียวจากด้านใน |
| 2 | **ปุ่มนิรภัย (Safety)** | กดปุ่ม Safety ร่วมกับปุ่มเปิด/ปิดเพื่อเปิดประตูเพื่อความปลอดภัย |
| 3 | **ปุ่มหมุนล็อก/ปลดล็อกด้วยมือ** | หมุนเพื่อล็อกหรือปลดล็อกประตูด้วยตนเองจากด้านใน |
| 4 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 5 | **สลักล็อก (Deadbolt)** | สลักล็อกหลักของประตูเมื่อประตูล็อก |
| 6 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |
| 7 | **แบตเตอรี่อัลคาไลน์ AA** | แหล่งพลังงานหลักสำหรับตัวล็อกประตูดิจิทัล |
| 8 | **สกรูยึดฝาครอบแบตเตอรี่** | ขันสกรูเพื่อยึดฝาครอบแบตเตอรี่ให้แน่น |

## อุปกรณ์ในกล่อง
- คู่มือการใช้งาน
- แผ่นทาบเจาะช่องติดตั้ง
- แบตเตอรี่ LR6 AA 4 ก้อน
- ชุดสกรู
- บัตร RFID (สติกเกอร์) 2 ใบ

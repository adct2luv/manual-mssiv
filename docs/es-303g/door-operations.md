# ขั้นตอนการเปิด/ปิดประตู

## การเปิดประตูจากด้านนอก

### เปิดด้วยรหัส PIN

1. สัมผัสคีย์แพดตัวเลข
2. ป้อนรหัส PIN แล้วตามด้วยปุ่ม [✱]
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

### เปิดด้วยฟังก์ชันสุ่มตัวเลข

ป้อนรหัสสุ่มใดๆ + ป้อนรหัส PIN + ปุ่ม [✱] หรือ ป้อนรหัส PIN + ป้อนรหัสสุ่มใดๆ + ปุ่ม [✱] ไม่ว่าจะวิธีใด เมื่อเสียงเพลงเมโลดี้ดังขึ้น ประตูจะเปิดออก

1. ป้อนรหัสสุ่มใดๆ + ป้อนรหัส PIN + ปุ่ม [✱] หรือ ป้อนรหัส PIN + ป้อนรหัสสุ่มใดๆ + ปุ่ม [✱] ไม่ว่าจะวิธีใด เมื่อเสียงเพลงเมโลดี้ดังขึ้น ประตูจะเปิดออก

:::caution[ข้อควรระวัง]
เมื่อเปิดระบบสัมผัสหลายจุด (Multi-touch) หลังจากป้อนรหัส PIN แล้ว ตัวเลขสุ่ม 3 หลักจะแสดงบนหน้าจอ ให้กดตัวเลขทั้ง 3 หลักตามด้วยปุ่ม [✱] ประตูจึงจะเปิด เมื่อเปิดระบบ Multi-touch ฟังก์ชันรหัสสุ่มปกติจะถูกปิดใช้งาน โดยฟังก์ชันรหัสสุ่มปกติจะถูกเปิดใช้งานเป็นค่าเริ่มต้น
:::

:::note[หมายเหตุ]
ฟังก์ชันรหัสสุ่มคือการสุ่มชุดตัวเลขเพื่อป้อนก่อนหรือหลังรหัส PIN จริง ใช้เพื่อป้องกันการล่วงรู้รหัสผ่าน โดยจำนวนหลักทั้งหมดรวมรหัสผ่านจริงแล้วต้องไม่เกิน 20 หลัก
:::

### เปิดด้วยบัตร RFID

1. ในโหมดตรวจจับอัตโนมัติ: ทาบบัตร RFID ที่ลงทะเบียนไว้บนตัวเครื่องอ่านสัญญาณ เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก
2. ในโหมดตรวจจับด้วยตนเอง: สัมผัสคีย์แพดตัวเลขเพื่อให้หน้าจอสว่างขึ้น จากนั้นทาบบัตร RFID ที่ลงทะเบียนบนตัวเครื่องอ่านสัญญาณ เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก

## การปิดประตูจากด้านนอก

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

1. สัมผัสคีย์แพดตัวเลขแล้วประตูจะล็อก (การตั้งค่า: ป้อนรหัส PIN + กดปุ่ม [✱] 3 ครั้ง) → [1]
2. กดปุ่ม [✱] แล้วประตูจะล็อก (การตั้งค่า: ป้อนรหัส PIN + กดปุ่ม [✱] 3 ครั้ง) → [2]

## การเปิดประตูจากด้านใน

### โหมดปกติ

1. เมื่อประตูล็อกอยู่ ให้กดปุ่ม [Open/Close] ประตูจะเปิดออก

### โหมดล็อกนิรภัย

1. เมื่อประตูล็อกอยู่ ให้กดปุ่ม [Safety] จากนั้นกดปุ่ม [Open/Close] ประตูจะเปิดออก

### โหมดล็อกด้วยตนเอง

1. เมื่อประตูล็อกอยู่ ให้หมุนลูกบิดล็อกด้วยตนเอง [Manual Lock Knob] ไปที่ทิศทางเปิด [Open] ประตูจะเปิดออก

## การปิดประตูจากด้านใน

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติ 2 วินาทีหลังจากประตูปิดลง

### โหมดล็อกด้วยตนเอง

1. หลังจากประตูปิดแล้ว ให้กดปุ่ม [Open/Close] ประตูจะล็อก

### การปิดประตูด้วยตนเอง

1. หลังจากประตูปิดแล้ว ให้หมุนลูกบิดล็อกด้วยตนเอง [Manual Lock Knob] ไปที่ทิศทางปิด [Close] ประตูจะล็อก

# การใช้งานเปิด-ปิดประตู

## วิธีเปิดประตูจากด้านนอก

### วิธีเปิดด้วยรหัสผ่าน (PIN)

1. แตะแป้นคีย์แพดตัวเลขเพื่อปลุกหน้าจอ
2. ป้อนรหัส PIN แล้วกดปุ่ม [✱]
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

### วิธีเปิดด้วยฟังก์ชันสุ่มตัวเลขความปลอดภัย (Random Number)

(Before) Enter any random number + ป้อนรหัส PIN + [✱] button. (After) ป้อนรหัส PIN + Enter any random number + [✱] button. ไม่ว่าอย่างไรก็ตาม เมื่อได้ยินเสียงเมโลดี้ ประตูจะเปิดออก

:::note
ฟังก์ชันสุ่มตัวเลขความปลอดภัย (Random Number) is the random set of numbers that is added before or after the original pin number is entered. It is used to prevent the pin number from being exposed. Including the pin number up to 20 digits only can be entered During authentication with random number.
:::

:::caution
When Multi-touch feature is enabled. After entering pin number, 3 digit random number will be displayed on the screen, press the 3 numbers, followed by [✱] button and ประตูจะเปิดออก. When Multi-touch feature is enabled. The "ฟังก์ชันสุ่มตัวเลขความปลอดภัย (Random Number)" is disabled. The "ฟังก์ชันสุ่มตัวเลขความปลอดภัย (Random Number)" is enabled in default upon purchase.
:::

### วิธีเปิดด้วยคีย์การ์ด RFID

1. เมื่ออยู่ในโหมดตรวจจับคีย์การ์ดอัตโนมัติ: ทาบคีย์การ์ด RFID ที่ลงทะเบียนแล้วบนพื้นที่เครื่องอ่านการ์ด จะมีเสียงสัญญาณเมโลดี้ดังขึ้นและประตูจะเปิดออก
2. เมื่ออยู่ในโหมดตรวจจับคีย์การ์ดด้วยตนเอง: แตะคีย์แพดตัวเลขด้านนอกเพื่อให้แป้นตัวเลขสว่างขึ้น จากนั้นทาบคีย์การ์ด RFID ที่ลงทะเบียนแล้วบนพื้นที่เครื่องอ่านการ์ด จะมีเสียงสัญญาณเมโลดี้ดังขึ้นและประตูจะเปิดออก


## วิธีปิดประตูจากด้านนอก

### ระบบล็อกอัตโนมัติ (Automatic Lock)

ประตูจะล็อกโดยอัตโนมัติ. After 2 seconds of closing the door.

### ระบบล็อกด้วยตนเอง (Manual Lock)

1. แตะแป้นคีย์แพดตัวเลข และ ประตูจะปิดล็อก. Setting: ป้อนรหัส PIN + กดปุ่ม [✱] 3 ครั้ง → 1
2. กดปุ่ม [✱] and ประตูจะปิดล็อก. Setting: ป้อนรหัส PIN + กดปุ่ม [✱] 3 ครั้ง → 2


## วิธีเปิดประตูจากด้านใน

### โหมดธรรมดา

1. ในขณะที่ประตูปิดล็อกอยู่ กดปุ่ม [Open/Close] ด้านใน ประตูจะเปิดออก

### โหมดล็อกนิรภัย (Safety Lock)

1. ในขณะที่ประตูปิดล็อกอยู่ กดปุ่ม [Safety] จากนั้นกดปุ่ม [Open/Close] ด้านใน ประตูจะเปิดออก

### ระบบล็อกด้วยตนเอง (Manual Lock)

1. ในขณะที่ประตูปิดล็อกอยู่ หมุนลูกบิดล็อกด้านในไปที่ตำแหน่งปลดล็อก [Open] ประตูจะเปิดออก


## วิธีปิดประตูจากด้านใน

### ระบบล็อกอัตโนมัติ (Automatic Lock)

The door will automatically lock. After 2 seconds of closing the door.

### ระบบล็อกด้วยตนเอง (Manual Lock)

1. หลังจากปิดประตูสนิท กดปุ่ม [Open/Close] ด้านใน ประตูจะล็อก

### การปิดล็อกด้วยตนเองด้วยลูกบิดแมนนวล

1. หลังจากปิดประตูสนิท หมุนลูกบิดล็อกด้านในไปที่ตำแหน่งล็อก [Close] ประตูจะล็อก

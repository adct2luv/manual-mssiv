# คุณสมบัติพิเศษ

## การลงทะเบียนรหัส PIN สำหรับผู้มาเยือน (Guest PIN)

### ขั้นตอน

1. กดปุ่ม [Registration] 1 ครั้ง ป้อนรหัส PIN ปัจจุบันแล้วตามด้วยปุ่ม [*]
2. กดปุ่มหมายเลข [3]
3. ป้อนรหัส PIN สำหรับผู้มาเยือนที่ต้องการลงทะเบียน (4~12 หลัก) จากนั้นกดปุ่ม [✱] บนคีย์แพดตัวเลข จะมีเสียงเพลงเมโลดี้ดังขึ้นและการลงทะเบียนเสร็จสมบูรณ์

:::note[หมายเหตุ]
หากยกเลิกโหมดการตั้งค่า รหัส PIN เดิมจะถูกกู้คืนและนำกลับมาใช้งานปกติ
:::

:::note[หมายเหตุ]
การลงทะเบียนใหม่จะทำการลบรหัส PIN เดิมที่เคยลงทะเบียนไว้
:::

:::caution[ข้อควรระวัง]
รหัส PIN สำหรับผู้มาเยือนใช้สำหรับผ่านเข้าประตูได้เพียง 1 ครั้งเท่านั้น
:::

## โหมดล็อกบังคับจากด้านนอก (Outside Force Lock)

ระบบความปลอดภัยนี้จะปิดการใช้งานการสั่งปลดล็อกผ่านปุ่ม [Open/Close] ที่ตัวเครื่องด้านในห้อง

### ขั้นตอน

1. ในขณะที่ประตูปิดสนิท ให้สัมผัสแป้นหน้าจอตัวเลข ป้อนรหัส PIN ที่ลงทะเบียนไว้
2. กดปุ่ม [*] แล้วตามด้วยปุ่ม [0]

**การยกเลิก:** เปิดประตูโดยใช้ก้านหมุนล็อก/ปลดล็อก และการตั้งค่าโหมดล็อกบังคับจะถูกยกเลิก

## โหมดล็อกบังคับจากด้านใน (Inside Force Lock)

โหมดความปลอดภัยนี้จะไม่ยินยอมให้มีการปลดล็อกประตูจากภายนอกทุกช่องทาง (ทั้งรหัส PIN และบัตร RFID)

### ขั้นตอน

1. ในขณะที่ประตูปิดสนิท ให้กดปุ่ม [Open/Close] ที่ตัวล็อกด้านในค้างไว้เป็นเวลา 3 วินาที เมื่อมีเสียงสัญญาณยืนยันดังขึ้น การตั้งค่าล็อกสองชั้นจะเสร็จสมบูรณ์

**การยกเลิก:** เมื่อเปิดประตูโดยใช้ปุ่มบิดล็อกด้วยตนเอง หรือกดปุ่ม [Open/Close] ด้านใน ระบบจะยกเลิกการตั้งค่าล็อกบังคับ (Force Lock)

## การตั้งค่าระดับเสียงการทำงาน

ฟังก์ชันนี้ช่วยให้ผู้ใช้งานสามารถปรับเปลี่ยนระดับเสียงการทำงานของระบบล็อกทีละระดับ (โปรดดำเนินการในขณะที่ประตูปิดสนิท)

### ขั้นตอน

1. กดหน้าจอคีย์แพดภายนอก ป้อนรหัส PIN แล้วกดปุ่ม [*]
2. กดปุ่ม [3] เพื่อเพิ่มระดับเสียง กดปุ่ม [6] เพื่อลดระดับเสียง

:::note[หมายเหตุ]
สามารถปรับระดับเสียงการทำงานได้สูงสุด 7 ระดับ
:::

## การตั้งค่าปิดเสียงชั่วคราว (Sound Mute)

สัมผัสหน้าจอคีย์แพดภายนอก กดปุ่ม [*] จากนั้นป้อนรหัส PIN การป้อนรหัสผ่านและการทำงานของตัวล็อกครั้งนี้จะเงียบสนิทไม่มีเสียงใดๆ (เสียงกดปุ่มและเสียงบิดล็อกจะถูกปิดชั่วคราว) โดยฟังก์ชันปิดเสียงนี้ใช้ได้สำหรับครั้งเดียวเท่านั้น

:::note[หมายเหตุ]
เมื่อเปิดใช้ฟังก์ชันปิดเสียงชั่วคราว หน้าจอคีย์แพดตัวเลขจะกะพริบเมื่อสัมผัสปุ่มกดเพื่อป้อนตัวเลข ระบบนี้จะทำงานสำหรับการเปิดประตูด้านนอกเพียงครั้งเดียวเท่านั้น หลังจากการใช้งานเสร็จสิ้นเสียงระบบทำงานจะกลับมาเป็นปกติ
:::

## การตั้งค่าโหมดล็อกอัตโนมัติ / ล็อกด้วยตนเอง

ตั้งค่าระบบล็อกประตูเป็นโหมดล็อกอัตโนมัติหรือล็อกด้วยตนเอง

### โหมดล็อกอัตโนมัติ

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 2 ครั้ง → [4]

### โหมดล็อกด้วยตนเอง

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 2 ครั้ง → [7]


:::note[หมายเหตุ]
ในโหมดล็อกด้วยตนเอง ประตูจะไม่ล็อกโดยอัตโนมัติหลังจากประตูปิดลง หากต้องการล็อกประตูจากด้านใน ให้กดปุ่ม [Open/Close] หรือสัมผัสหน้าจอคีย์แพดตัวเลขด้านนอกแล้วกดปุ่ม [*] เพื่อล็อกประตู
:::

:::note[หมายเหตุ]
เมื่อซื้อสินค้า โหมดล็อกด้วยตนเองจะถูกตั้งค่าเป็นค่าเริ่มต้น
:::

## การสแกนบัตร RFID อัตโนมัติ / ด้วยตนเอง

ตั้งค่าโหมดการสแกนบัตร RFID เป็นแบบอัตโนมัติหรือแบบตรวจสอบสิทธิ์หน้าจอ

### การตั้งค่าตรวจจับการ์ดอัตโนมัติ

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 2 ครั้ง → [1]

### การตั้งค่าตรวจจับการ์ดแบบด้วยตนเอง

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 2 ครั้ง → [3]


:::note[หมายเหตุ]
ในโหมดด้วยตนเอง: สัมผัสหน้าจอคีย์แพดตัวเลข จากนั้นทาบบัตร RFID บนตัวเครื่องอ่านเพื่อเปิดประตู
:::

:::note[หมายเหตุ]
ในโหมดอัตโนมัติ: ทาบบัตร RFID บนตัวเครื่องอ่านเพื่อปลดล็อกเปิดประตู
:::

:::note[หมายเหตุ]
โหมดอัตโนมัติถูกเปิดใช้งานเป็นค่าเริ่มต้น
:::

## การตั้งค่าโหมดเปิดประตูแบบปกติและโหมดนิรภัย

เมื่อคุณซื้อผลิตภัณฑ์เป็นครั้งแรก ตัวล็อกจะถูกตั้งค่าการทำงานไว้ที่โหมดปกติเริ่มต้น

### การตั้งค่าโหมดปกติ (Normal Mode)

คุณสามารถปลดล็อกเปิดประตูได้ง่ายๆ เพียงกดปุ่ม [OPEN / CLOSE] ที่ตัวเครื่องด้านในห้อง

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 3 ครั้ง → [4]

### การตั้งค่าโหมดนิรภัย (Safe Mode)

คุณสามารถเปิดประตูได้โดยการกดปุ่ม [Force Lock] จากนั้นกดปุ่ม [OPEN / CLOSE] ตามลงไปทันทีภายในระยะเวลา 1 วินาที

1. สัมผัสคีย์แพดตัวเลข → ป้อนรหัส PIN → กดปุ่ม [✱] 3 ครั้ง → [5]


:::note[หมายเหตุ]
การกดปุ่ม [Force Lock] และปุ่ม [OPEN / CLOSE] ต่อเนื่องกัน ช่วยเพิ่มประสิทธิภาพความปลอดภัยของฟังก์ชันการปลดล็อกภายใน
:::

## การตั้งค่าระบบสัมผัสหลายจุด (Multi-touch)

ฟังก์ชันความปลอดภัย Multi-Touch ช่วยให้ผู้ใช้สามารถใส่ตัวเลขเพิ่มเติมหลังจากป้อนรหัส PIN เพื่อช่วยลบรอยนิ้วมือบนแผงปุ่มกด และป้องกันไม่ให้รหัส PIN ถูกแอบมอง

**การยกเลิก:** หากต้องการยกเลิกการตั้งค่า ให้ทำซ้ำตามขั้นตอนวิธีการเดิมอีกครั้งระบบจะปิดการทำงานลง

:::note[หมายเหตุ]
เมื่อเปิดใช้ระบบความปลอดภัย Multi-Touch หลังจากกดปุ่มรหัส PIN จะปรากฏตัวเลขสุ่ม 3 ตัวขึ้นมาบนหน้าจอ ให้ป้อนตัวเลขสุ่ม 3 ตัวนั้นแล้วกดปุ่ม [✱] เพื่อปลดล็อกเข้าประตู
:::

:::note[หมายเหตุ]
เมื่อเปิดใช้ระบบความปลอดภัย Multi-Touch ระบบจะปิดการทำงานของระบบสุ่มตัวเลขแบบอื่น และหากสุ่มตัวเลขทำงานอยู่ ระบบ Multi-Touch จะถูกปิดการทำงาน
:::

## การตั้งค่าระบบยืนยันตัวตนสองชั้น

### ขั้นตอน

1. กดปุ่ม [Registration] ค้างไว้ 5 วินาที ไฟ LED ทั้งหมดจะสว่างขึ้น หลังจาก 3 วินาทีเสียงเตือนจะหยุด และอีก 2 วินาทีไฟจะเปิดขึ้นมาอีกครั้ง
2. ป้อนรหัสผ่านปัจจุบัน เมื่อคุณกดปุ่ม [✱] ไฟ LED แผงด้านหน้าจะกะพริบและสว่างขึ้น
3. กดปุ่ม [1] เพื่อปิดการทำงานระบบความปลอดภัยสองชั้น กดปุ่ม [3] เพื่อเปิดการทำงานระบบความปลอดภัยสองชั้น
4. ในระหว่างการตั้งค่าระบบยืนยันตัวตนสองชั้น เมื่อได้ยินเสียงยืนยัน หน้าจอและไฟแสดงสถานะทั้งหมดจะดับลง

**วิธีเปิดประตู:** รหัส PIN + บัตร RFID

:::note[หมายเหตุ]
ต้องทำการลงทะเบียนรหัส PIN และบัตรสมาร์ทคีย์ก่อนเปิดใช้งานฟังก์ชันนี้
:::

## การตั้งค่าเสียงเพลงเมโลดี้

### ขั้นตอน

1. สัมผัสคีย์แพดตัวเลข ป้อนรหัส PIN แล้วกดปุ่ม [*] 3 ครั้ง
2. เลือกจากเมนู — กดปุ่ม [7] เพื่อปิดเสียงเพลงเมโลดี้ กดปุ่ม [8] เพื่อลดเสียง และกดปุ่ม [9] เพื่อเพิ่มเสียง

:::note[หมายเหตุ]
ฟังก์ชันนี้จะทำงานเฉพาะในโหมดล็อกด้วยตนเองเท่านั้น
:::

:::note[หมายเหตุ]
หลังจากได้ยินเสียงเพลงเมโลดี้แล้ว หากยังไม่ได้ปิดประตูแต่มีการใช้งานฟังก์ชันใหม่ เสียงเพลงเมโลดี้จะปิดลง
:::

:::note[หมายเหตุ]
หลังจากประตูเปิดแล้ว จะไม่ได้ยินเสียงเพลงเมโลดี้
:::

## การตั้งค่าโมดูลระบบเครือข่ายในบ้านและรีโมทคอนโทรล

### ขั้นตอนการลงทะเบียน

1. กดปุ่ม [Registration] ป้อนรหัส PIN แล้วตามด้วยปุ่ม [*]
2. การลงทะเบียนโมดูลเครือข่ายในบ้าน / รีโมทคอนโทรล: กดปุ่ม [Registration] / กดปุ่ม [Open] บนรีโมทคอนโทรล
3. สำหรับการลงทะเบียนรีโมทคอนโทรลเพิ่มเติม ให้ทำซ้ำขั้นตอนที่ 2

### ขั้นตอนการลบข้อมูล

1. กดปุ่ม [Registration] ป้อนรหัส PIN แล้วตามด้วยปุ่ม [*]
2. กดปุ่ม [0]
3. กดปุ่ม [*] ค้างไว้ 5 วินาที เมื่อได้ยินเสียงเพลงเมโลดี้ดังขึ้น การลบเสร็จสมบูรณ์

:::note[หมายเหตุ]
สามารถลงทะเบียนรีโมทคอนโทรลได้สูงสุด 5 อัน
:::

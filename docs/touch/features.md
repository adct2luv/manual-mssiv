# คุณสมบัติพิเศษ

## การลงทะเบียนรหัส PIN สำหรับผู้มาเยือน (Guest PIN)

### ขั้นตอน

1. เปิดฝาครอบแบตเตอรี่ (ตัวเครื่องด้านใน) จากนั้นกดปุ่ม [Registration] 1 ครั้ง จะมีเสียงสัญญาณบี๊บดังขึ้น
2. ป้อนรหัส PIN ปัจจุบันตามด้วยปุ่ม [✱]
3. ตรวจสอบว่ามีไฟสว่างที่ตัวเลข [1, 2, 3, 7, 8, 9, 0, #] บนคีย์แพดตัวเลข จากนั้นกดปุ่มหมายเลข [3]
4. ป้อนรหัส PIN สำหรับผู้มาเยือนที่ต้องการลงทะเบียน (4~12 หลัก) จากนั้นกดปุ่ม [*] เพื่อเสร็จสิ้นขั้นตอนการลงทะเบียน

:::note[หมายเหตุ]
รหัส PIN สำหรับผู้มาเยือนใช้สำหรับผ่านเข้าประตูได้เพียง 1 ครั้งเท่านั้น
:::

:::note[หมายเหตุ]
การตั้งค่ารหัส PIN สำหรับผู้มาเยือนใหม่จะทำการลบรหัสผ่านเก่าออก
:::

:::note[หมายเหตุ]
หากยกเลิกโหมดการตั้งค่า รหัส PIN เดิมจะถูกกู้คืนและนำกลับมาใช้งานปกติ
:::

## outside_force_lock

เมื่อตั้งค่าโหมดนี้แล้ว จะส่งผลให้ไม่สามารถกดสั่งเปิดล็อกประตูจากปุ่ม [Open/Close] ด้านในได้

### ขั้นตอน

1. ในขณะที่ประตูปิดอยู่ ให้สัมผัสแป้นคีย์แพดตัวเลข
2. ป้อนรหัส PIN
3. กดปุ่ม [*] แล้วตามด้วยปุ่ม [0]
4. เมื่อได้ยินเสียงเพลงเมโลดี้ดังขึ้นเสร็จสิ้นขั้นตอนการตั้งค่า (โดยจะปิดใช้งานการกดปุ่ม [Open/Close] ที่ตัวล็อกฝั่งในห้อง)

**การยกเลิก:** เมื่อปลดล็อกและเปิดประตูด้วยรหัส PIN ระบบจะยกเลิกการตั้งค่าโหมดล็อกบังคับ (Force Lock) ทันที

## internal_force_lock

เมื่อเปิดใช้การล็อกบังคับจากด้านใน จะไม่ยินยอมให้เข้าประตูจากภายนอกทุกวิธี แม้แต่การป้อนรหัส PIN ก็จะปิดการใช้งาน

### ขั้นตอน

1. ในขณะที่ประตูปิดอยู่
2. กดปุ่ม [Force Lock] ค้างไว้ 3 วินาที เมื่อเสียงเพลงเมโลดี้ดังขึ้น การตั้งค่าเสร็จสิ้น

**การยกเลิก:** เปิดประตูด้วยลูกบิดบิดด้วยมือ [Manual Lock Knob] หรือกดปุ่ม [Open/Close] ด้านใน โหมดล็อกบังคับ [Force Lock] จะถูกยกเลิก

## โหมดล็อกอัตโนมัติ / ล็อกด้วยตนเอง

สลับโหมดล็อกประตูเป็นโหมดอัตโนมัติหรือโหมดล็อกด้วยตนเอง

### ขั้นตอน

1. กดปุ่ม [Open/Close]
2. สัมผัสคีย์แพดตัวเลข
3. ป้อนรหัส PIN จากนั้นกดปุ่ม [✱] 2 ครั้ง
4. กดปุ่ม [4]: ตั้งค่าล็อกอัตโนมัติ กดปุ่ม [7]: ตั้งค่าล็อกด้วยตนเอง

:::note[หมายเหตุ]
โหมดล็อกอัตโนมัติจะถูกตั้งค่าเป็นค่าเริ่มต้น เมื่อคุณตั้งค่าล็อกด้วยตนเอง ประตูจะไม่ล็อกอัตโนมัติหลังจากประตูปิด คุณต้องกดปุ่ม [Open/Close] ด้านใน หรือสัมผัสคีย์แพดตัวเลขด้านนอกเพื่อล็อกประตู
:::

## ฟังก์ชันสัมผัสหลายจุด (Multi-Touch)

### ขั้นตอน

1. สัมผัสคีย์แพดตัวเลขที่แผงตัวเครื่องด้านนอก ป้อนรหัส PIN แล้วตามด้วยปุ่ม [*]
2. กดปุ่ม [5] บนคีย์แพดตัวเลข จะได้ยินเสียงเพลงเมโลดี้และสิ้นสุดการตั้งค่าฟังก์ชัน Multi-Touch

**การยกเลิก:** หากต้องการยกเลิกการตั้งค่า ให้ทำซ้ำตามขั้นตอนวิธีการเดิมอีกครั้งระบบจะปิดการทำงานลง

:::note[หมายเหตุ]
ฟังก์ชันความปลอดภัย Multi-Touch ช่วยปกป้องรหัสผ่านของคุณจากการถูกเปิดเผยหรือคาดเดาจากร่องรอยนิ้วมือที่หลงเหลือบนแผงหน้าจอคีย์แพดตัวเลข โดยฟังก์ชันนี้จะถูกปิดการทำงานเป็นค่าเริ่มต้น
:::

:::caution[ข้อควรระวัง]
เมื่อเปิดใช้ระบบความปลอดภัย Multi-Touch หลังจากกดปุ่มรหัส PIN จะปรากฏตัวเลขสุ่ม 3 ตัวนั้นแล้วกดปุ่ม [*] เพื่อปลดล็อกเข้าประตู ซึ่งระบบนี้จะช่วยปิดการทำงานระบบความปลอดภัยสุ่มตัวเลขแบบอื่นโดยอัตโนมัติ
:::

## ระบบยืนยันตัวตนสองชั้น

### ขั้นตอน

1. กดปุ่ม [Registration] ค้างไว้ 5 วินาที ไฟ LED ทั้งหมดจะสว่างขึ้น หลังจาก 3 วินาทีเสียงเตือนจะหยุด และอีก 2 วินาทีไฟจะเปิดขึ้นมาอีกครั้ง
2. ป้อนรหัส PIN ปัจจุบัน เมื่อคุณกดปุ่ม [✱] ไฟ LED แผงด้านหน้าจะกะพริบและสว่างขึ้น
3. กดปุ่ม [1] เพื่อปิดโหมดยืนยันตัวตนสองชั้น หรือกดปุ่ม [3] เพื่อเปิดโหมด เมื่อได้ยินเสียงยืนยัน หน้าจอและระบบจะปิดลงเสร็จสมบูรณ์

:::note[หมายเหตุ]
ต้องทำการลงทะเบียนรหัส PIN และการ์ด RFID ก่อนเปิดใช้งานฟังก์ชันนี้
:::

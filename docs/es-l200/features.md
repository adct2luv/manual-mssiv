# คุณสมบัติพิเศษ

## ฟังก์ชันสุ่มตัวเลขความปลอดภัย

เพิ่มตัวเลขสุ่มก่อนหรือหลังรหัส PIN จริงของคุณ เพื่อป้องกันคนแอบดูแป้นกด รวมรหัสผ่านและเลขสุ่มได้สูงสุด 20 หลัก

## ความปลอดภัยแบบ Multi-Touch

เมื่อกดรหัสผ่านเสร็จ หน้าจอจะแสดงตัวเลขสุ่มขึ้นมา 3 หลัก ให้กดตัวเลขเหล่านั้นเพื่อลบรอยนิ้วมือบนแผงคีย์แพด ป้องกันแป้นพิมพ์มีคราบรอยกดนิ้วมือ

## ระบบล็อกประตู อัตโนมัติ/ด้วยตัวเอง

สลับระหว่างโหมดล็อกอัตโนมัติ (ล็อก 2 วินาทีหลังจากประตูปิดลงสนิท) และโหมดแมนนวลล็อก (แตะคีย์แพดภายนอกหรือบิดลูกบิดด้านในเองเพื่อล็อก)

## การปรับระดับเสียงล็อกประตู

ปรับระดับเสียงการทำงานได้ 7 ระดับ โดยตั้งค่าผ่านแป้นพิมพ์ตัวเลขด้านนอก

## ปิดเสียงชั่วคราว (Mute)

ปิดเสียงการทำงานชั่วคราวในการปลดล็อกครั้งถัดไปโดยเริ่มกดปุ่ม [*] ก่อนกดรหัสผ่าน เหมาะสำหรับเปิดห้องเวลากลางคืนโดยไม่รบกวนผู้อื่น

## โหมดตรวจจับคีย์การ์ด อัตโนมัติ/ด้วยตนเอง

ในโหมดออโต้ คุณสามารถทาบบัตรเปิดประตูได้ทันที ในโหมดแมนนวลคุณต้องแตะคีย์แพดเพื่อปลุกหน้าจอก่อนจะสามารถทาบบัตรได้

## ระบบยืนยันตัวตนสองชั้นเพื่อความปลอดภัยสูงสุด

ตั้งค่าระบบล็อกให้ตรวจจับการยืนยันตัวตนแบบควบคู่กัน (รหัสผ่าน PIN ร่วมกับ บัตร RFID หรือลายนิ้วมือ) จึงจะปลดล็อกประตูได้สำเร็จ

# ขั้นตอนการเปิด/ปิดประตู

## การเปิดประตูจากด้านนอก

### เปิดด้วยรหัส PIN

1. แตะแป้นคีย์แพดตัวเลขเพื่อเปิดไฟหน้าจอ
2. ป้อนรหัส PIN ที่ลงทะเบียนไว้ (ค่าเริ่มต้นคือ 1, 2, 3, 4)
3. กดปุ่ม [✱]
4. ผลักหรือดึงประตูเพื่อเปิด

### เปิดด้วยบัตร RFID / คีย์แทก

1. ทาบบัตร RFID หรือคีย์แทกที่ลงทะเบียนไว้บนเครื่องอ่านการ์ด
2. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะปลดล็อก
3. ผลักหรือดึงประตูเพื่อเปิด

### เปิดด้วยลายนิ้วมือ

1. แตะหรือวางนิ้วมือที่ลงทะเบียนไว้บนเครื่องอ่านลายนิ้วมือ
2. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะปลดล็อก
3. ผลักหรือดึงประตูเพื่อเปิด

## การปิดประตูจากด้านนอก

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

ในโหมดล็อกด้วยตนเอง ประตูจะไม่ล็อกโดยอัตโนมัติเมื่อปิด

1. ปิดประตูให้สนิท
2. สัมผัสคีย์แพดตัวเลขด้านนอก หรือกดปุ่ม [✱] เพื่อทำการล็อกประตู

## การเปิดประตูจากด้านใน

### เปิดด้วยปุ่ม Open/Close

1. ในขณะที่ประตูล็อกอยู่ ให้กดปุ่ม [Open/Close] ที่ตัวเครื่องด้านใน
2. ประตูจะปลดล็อก จากนั้นผลักหรือดึงเพื่อเปิดประตู

### เปิดด้วยปุ่มบิดล็อกด้วยตนเอง

1. บิดหมุนปุ่มบิดล็อก/ปลดล็อกด้วยตนเอง (Manual Lock/Unlock Knob) ไปที่ทิศทางเปิด [Open]
2. ประตูจะปลดล็อก จากนั้นผลักหรือดึงเพื่อเปิดประตู

## การปิดประตูจากด้านใน

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

ในโหมดล็อกด้วยตนเอง ประตูจะไม่ล็อกโดยอัตโนมัติเมื่อปิด

1. ปิดประตูให้สนิท
2. กดปุ่ม [Open/Close] หรือหมุนปุ่มบิดล็อกด้านในเพื่อทำการล็อกประตู

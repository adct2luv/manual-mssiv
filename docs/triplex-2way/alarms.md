# สัญญาณเตือนและความปลอดภัย

## สัญญาณเตือนป้องกันการเดารหัสผ่าน

การป้อนรหัส PIN ผิดติดต่อกัน 5 ครั้ง จะส่งผลให้สัญญาณเตือนภัยดังขึ้นและระบบล็อกจะไม่ทำงานเป็นเวลา 1 นาที

## deadbolt_error

เมื่อประตูปิดสนิทไม่เรียบร้อยหรือสลักล็อกประตูด้านใน (Deadbolt) มีความติดขัดทำงานไม่สะดวก ระบบจะส่งสัญญาณเสียงแจ้งเตือนข้อผิดพลาด 3 ครั้งติดต่อกัน

## สัญญาณเตือนเปลี่ยนแบตเตอรี่

หากมีเสียงเตือนดังขึ้นในระหว่างการใช้งานปกติ แสดงว่าถึงเวลาที่ต้องเปลี่ยนแบตเตอรี่ โปรดเปลี่ยนแบตเตอรี่ใหม่ทั้งหมดในเวลานี้

## แบตเตอรี่ฉุกเฉิน

### ขั้นตอน

1. แบตเตอรี่ 9V ต้องซื้อแยกต่างหาก
2. แตะขั้วแบตเตอรี่ 9V เข้ากับขั้วสัมผัสแบตเตอรี่ฉุกเฉินที่แผงตัวเครื่องด้านหน้า
3. กดปุ่ม [✱]
4. ป้อนรหัส PIN
5. กดปุ่ม [*] อีกครั้งเพื่อเปิดประตู

# ส่วนประกอบ

## ตัวล็อกด้านนอก

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **แผงคีย์แพดตัวเลข** | ป้อนรหัส PIN หรือรหัสตั้งค่าการทำงาน (1-9, *, 0, #) |
| 2 | **ไฟแจ้งเตือนสถานะแบตเตอรี่** | ไฟเตือนจะสว่างขึ้นเมื่อแบตเตอรี่อ่อน |
| 3 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 4 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |
| 5 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ปุ่มเปิด/ปิดประตู (Open/Close)** | กดปุ่มเพื่อล็อกหรือปลดล็อกประตูแบบสัมผัสเดียวจากด้านใน |
| 2 | **Internal force lock button** |  |
| 3 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 4 | **เซ็นเซอร์ล็อกอัตโนมัติ** | ตรวจจับเมื่อประตูปิดเพื่อสั่งการล็อกอัตโนมัติ |
| 5 | **สลักล็อก (Deadbolt)** | สลักล็อกหลักของประตูเมื่อประตูล็อก |
| 6 | **ปุ่มหมุนล็อก/ปลดล็อกด้วยมือ** | หมุนเพื่อล็อกหรือปลดล็อกประตูด้วยตนเองจากด้านใน |
| 7 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |

# คู่มือจัดการสายเคเบิลตัวล็อกด้านนอก

คู่มือนี้จัดทำขึ้นเพื่อเน้นย้ำถึงขั้นตอนสำคัญและคำเตือนในการจัดสายสัญญาณของตัวเครื่องด้านนอก (Outer Body) ขณะติดตั้งประกบเข้ากับตัวเครื่องด้านใน (Inner Body)

## ⚠️ ข้อควรระวังและผลกระทบของการจัดสายที่ผิดวิธี

:::caution[การจัดสายเคเบิลที่ถูกต้อง]
- เมื่อจัดเก็บสายสัญญาณของตัวเครื่องด้านนอกเข้าไปในช่องและยึดเข้ากับตัวเครื่องด้านใน **ต้องจัดเรียงสายอย่างเป็นระเบียบ** หลีกเลี่ยงไม่ให้สายไปทับหรือขวางชิ้นส่วนกลไก
- **หากจัดเก็บสายสัญญาณอย่างไม่ถูกต้อง (Incorrect Way):** สายไฟอาจจะถูกหนีบ กดทับ หรือเข้าไปขัดขวางกลไกการล็อกและมอเตอร์ของตัวเครื่องด้านใน (Inner Body) ทำให้ระบบปลดล็อกด้วยมอเตอร์ขัดข้อง ทำงานผิดพลาด หรือทำให้ชิ้นส่วนภายในเสียหายได้
- โปรดอ้างอิงภาพเปรียบเทียบในคู่มือประกอบเพื่อใช้ขั้นตอนการเดินสายที่ถูกต้อง (**Correct Way**) และใช้ความระมัดระวังสูงสุดในระหว่างการขันน็อตยึดตัวเครื่อง
:::

# คู่มือการใช้งาน IR Sensor สำหรับ Face ID

คู่มืออ้างอิงการใช้งานและการตั้งค่าระยะตรวจจับของเซ็นเซอร์ตรวจจับความเคลื่อนไหวอินฟราเรด (IR Sensor) สำหรับระบบปลดล็อกด้วยใบหน้า (Face ID) ในระบบดิจิทัลล็อกรุ่น **ES-S740D** และ **ES-S510D/H**

## 📡 ภาพรวมระบบตรวจจับบุคคลอัตโนมัติ (IR Sensor Overview)
ระบบเซ็นเซอร์อินฟราเรดช่วยตรวจจับวัตถุหรือบุคคลที่เดินเข้ามาใกล้ตัวล็อกเพื่อปลุกระบบจดจำใบหน้า (Face ID) ให้พร้อมใช้งานโดยอัตโนมัติโดยไม่ต้องสัมผัสแป้นพิมพ์

### ระยะตรวจจับตามสภาพแวดล้อมต่างๆ:
1. **เมื่อหันหน้าเข้าหากระจกห้องน้ำ (WC Mirror):**
   - **ระยะตรวจจับ:** ~1.75 เมตร
   - **ความสูงเฉลี่ยในการติดตั้งตรวจวัด:** 1.2 เมตร
   - **ผลลัพธ์:** เซ็นเซอร์ IR จะตรวจจับได้ไกลที่สุดเนื่องจากเป็นพื้นผิวสะท้อนแสงสูง
2. **ตรวจจับร่างกายบุคคล (Person):**
   - **ระยะตรวจจับ:** ~0.9 เมตร
   - **ผลลัพธ์:** ตรวจจับได้แม่นยำในระยะเหมาะสมเมื่อมีบุคคลเดินเข้ามาใกล้ในแนวตรง
3. **เมื่ออยู่ใกล้กระดานไวท์บอร์ดหรือแผ่นสะท้อนแสง (White Board / Reflection):**
   - **ระยะตรวจจับ:** ~1.25 เมตร
   - **ผลลัพธ์:** มีระยะการทำงานกว้างขึ้นเนื่องจากแรงสะท้อนของพื้นผิววัตถุสีขาวหรือผิวมันเงา

---

## ⚠️ การตรวจสอบสภาพแวดล้อมก่อนติดตั้ง

:::warning[คำเตือนเรื่องตำแหน่งการติดตั้ง]
- ผลการทดสอบข้างต้นเป็นเพียงค่าอ้างอิงในห้องปฏิบัติการ การใช้งานจริงจะขึ้นอยู่กับสภาพแวดล้อมและพื้นที่ในการติดตั้งประตู
- หากหน้าประตูของคุณหันเข้าหาผนังโดยตรง หรือทางเดินแคบ (ผนังชนผนัง) วัตถุสะท้อนแสงอาจทำให้ตัวล็อกตื่นเองบ่อยเกินไป
- ในกรณีดังกล่าว กรุณาเปลี่ยนการตั้งค่าปิดใช้งานเซ็นเซอร์อินฟราเรด (**IR Sensor ON/OFF setting**) หรือตั้งค่าตามวิธีที่ระบุไว้ในคู่มือประจำรุ่น
:::

# ขั้นตอนการเปิด/ปิดประตู

## การเปิดประตูจากด้านนอก

### เปิดด้วยรหัส PIN

1. สัมผัสคีย์แพดตัวเลข
2. ป้อนรหัส PIN แล้วตามด้วยปุ่ม [✱]
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

### เปิดด้วยฟังก์ชันสุ่มตัวเลข

เหมือนกับรุ่น EF-8000L: ป้อนรหัสสุ่มปะหน้าหรือปิดท้ายรหัส PIN จริง ความยาวรวมรหัสผ่านไม่เกิน 20 หลัก

### Opening by Smart Key

1. การตรวจจับอัตโนมัติ: ทาบบัตรสมาร์ทคีย์บนเครื่องอ่านสัญญาณ ประตูจะเปิดออก
2. การตรวจจับด้วยตนเอง: สัมผัสคีย์แพดตัวเลข จากนั้นทาบบัตรสมาร์ทคีย์บนเครื่องอ่านสัญญาณ ประตูจะเปิดออก

## การปิดประตูจากด้านนอก

### Auto

locks 2 sec after close

### Manual

touch number pad

## การเปิดประตูจากด้านใน

### เปิดด้วยมือจับประตูด้านใน

กดที่จับลงเพื่อเปิดประตู

## การปิดประตูจากด้านใน

### Manual

กดปุ่ม [Open/Close]

### Auto

locks automatically

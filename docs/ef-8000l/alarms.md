# สัญญาณเตือนฉุกเฉิน (Emergency Alarm Features)

## 1. สัญญาณเตือนไฟไหม้ (Fire Sensor Alarm)

เมื่อเซ็นเซอร์ตรวจจับอุณหภูมิสูงผิดปกติ (≥ 62°C ± 5°C) ระบบจะ:
- ส่งเสียงเตือนดัง
- **ปลดล็อกประตูอัตโนมัติ** เพื่อให้อพยพได้

:::caution
- การโดน **แสงแดดโดยตรงเป็นเวลานาน** อาจทำให้เซ็นเซอร์ทำงานผิดพลาดได้
- แม้อยู่ในโหมด "Double Lock" ประตูจะ **เปิดอัตโนมัติ** เมื่ออุณหภูมิสูงเกิน
- เมื่ออุณหภูมิกลับสู่ปกติ ล็อกจะกลับสู่สถานะทำงานปกติ
:::

## 2. สัญญาณเตือนการบุกรุก (Intrusion Alarm)

เมื่อล็อกตรวจพบการงัดแงะจากด้านในหรือด้านนอก ระบบจะส่งเสียงเตือนเป็นจังหวะ

**วิธียกเลิก:**
- ถอดแบตเตอรี่ออก
- หรือใส่ PIN / แตะบัตร / ลายนิ้วมือที่ลงทะเบียนไว้

:::note
ถ้าอุณหภูมิยังสูง (อยู่ในสถานะ Fire Alarm) จะยกเลิกสัญญาณ Intrusion ไม่ได้
:::

## 3. สัญญาณเตือนการแกล้ง + ล็อก 1 นาที (Anti-Prank Alarm & 1 Minute Lock)

เมื่อมีการใส่ **PIN / Smart Key / Fingerprint ผิด 5 ครั้งติดต่อกัน** ระบบจะ:
- ส่งเสียงเตือน
- **ล็อกล็อก 1 นาที** ไม่สามารถใช้งานได้

หลังจาก 1 นาที ล็อกจะกลับสู่สถานะปกติ

## 4. สัญญาณเตือนกลอนตายติด (Deadbolt Jammed Alarm)

ขณะเปิดหรือปิดประตู หาก **กลอนตายติดหรือวางไม่สมบูรณ์** ระบบจะส่งเสียง **"บี๊บ-บี๊บ-บี๊บ"** 3 ครั้ง

:::caution
หากเสียงเตือนยังคงดังอยู่หรือกลอนตายทำงานผิดปกติ กรุณาติดต่อช่างติดตั้งในพื้นที่ อาจมีค่าใช้จ่ายในการซ่อม
:::

## 5. สัญญาณเตือนเปลี่ยนแบตเตอรี่ (Replace Battery Alarm)

เมื่อใช้งานปกติ หาก **ไฟแบตเตอรี่ติด** พร้อมเสียง **"บี๊บ-รี๊-รี๊ป"** แสดงว่าแบตเตอรี่ใกล้หมด กรุณา **เปลี่ยนแบตเตอรี่ทั้งหมดทันที**

## 6. กุญแจกลไกฉุกเฉิน (Emergency Key) — เฉพาะรุ่นที่มี Emergency Key

ใช้เมื่อล็อกทำงานผิดปกติหรือแบตเตอรี่หมด

**ขั้นตอน:**
1. **เปิดฝาครอบรูกุญแจ** ที่ด้านบนของที่จับด้านนอก (ดันไปทางขวา)
2. **ใส่กุญแจฉุกเฉิน** เข้าไปในรูกุญแจ หมุนจาก **ขวาไปซ้าย**
3. เมื่อได้ยินเสียง **"ติ๊ก"** แสดงว่า **กลอนตายเลื่อนกลับสู่ตำแหน่งเดิม** ให้ดึงกุญแจออก
4. **หมุนที่จับลง** เพื่อเปิดประตู
5. **ดันฝาครอบรูกุญแจกลับ** เพื่อปิด

:::warning
เก็บกุญแจฉุกเฉินไว้ในที่ปลอดภัย เข้าถึงได้ แต่ไม่ใกล้ประตูหน้าบ้าน
:::

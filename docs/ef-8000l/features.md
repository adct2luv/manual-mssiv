# ฟีเจอร์เสริม (Convenient Features)

## 1. โหมดล็อกอัตโนมัติ/ด้วยตัวเอง (Auto/Manual Lock Setting)

ตั้งค่าให้ล็อกอัตโนมัติหลังประตูปิด หรือล็อกด้วยตัวเอง

**ขั้นตอน (ขณะประตูเปิดครึ่งหนึ่ง):**
1. กดปุ่ม **Manual Lock** เมื่อกลอนตายเลื่อนออก
2. แตะคีย์แพด
3. ใส่รหัส PIN แล้วกด `[*]` **2 ครั้ง**
4. กด **[4]** สำหรับ Auto Lock หรือ **[7]** สำหรับ Manual Lock

:::note
โหมด Auto เป็นค่าเริ่มต้น ในโหมด Manual ประตูจะไม่ล็อกอัตโนมัติ ต้องแตะคีย์แพดหรือกดปุ่ม Manual Lock ด้านใน
:::

## 2. โหมดตรวจจับ Smart Key อัตโนมัติ/ด้วยตัวเอง (Smart Key Auto/Manual Detection)

**ขั้นตอน:**
1. แตะคีย์แพด
2. ใส่รหัส PIN แล้วกด `[*]` **2 ครั้ง**
3. กด **[1]** สำหรับ Auto หรือ **[3]** สำหรับ Manual

:::note
- **Auto Mode**: วาง Smart Key บนเซ็นเซอร์ได้เลย
- **Manual Mode**: ต้องแตะคีย์แพดก่อน แล้ววาง Smart Key
- **Auto เป็นค่าเริ่มต้น**
:::

## 3. ปิดเสียงชั่วคราว (Sound Mute)

ปิดเสียงการทำงานสำหรับการใช้งานครั้งเดียว

**ขั้นตอน:**
1. แตะคีย์แพด แล้วกด `[*]` (ในสถานะ mute)
2. ใช้งานด้วย PIN หรือ Fingerprint

:::note
ใช้ได้ครั้งเดียว เสียงจะกลับมาในการใช้งานครั้งถัดไป
:::

## 4. ปรับระดับเสียง (Sound Volume) — 7 ระดับ

**ขั้นตอน:**
1. แตะคีย์แพด
2. ใส่รหัส PIN แล้วกด `[*]`
3. กด **[3]** เพื่อเพิ่มเสียง หรือ **[6]** เพื่อลดเสียง

:::note
- ปรับได้สูงสุด 7 ระดับ
- ในโหมด Mute คีย์แพดจะกระพริบเมื่อกดตัวเลข
- ระดับเสียงเริ่มต้น: สูง
:::

## 5. ระบบ Multi-touch Security

เพิ่มตัวเลขสุ่มก่อน/หลัง PIN เพื่อป้องกันรอยนิ้วมือบนคีย์แพด

**ขั้นตอน:**
1. แตะคีย์แพด
2. ใส่รหัส PIN แล้วกด `[*]`
3. กด **[5]** — จะได้ยินเพลงยืนยัน
4. ทำซ้ำขั้นตอน 1-3 เพื่อยกเลิก

:::caution เมื่อเปิด Multi-touch
หลังใส่ PIN คีย์แพดจะแสดงตัวเลขสุ่ม 3 ตัว ให้กดตัวเลขเหล่านั้นก่อนกด `[*]`
- เมื่อ Multi-touch เปิด → Random Number feature จะถูกปิด
- เมื่อ Random Number เปิด → Multi-touch feature จะถูกปิด
:::

## 6. ระบบ Dual-Mode Security (ต้องยืนยัน 2 ชั้น)

หลังใส่ PIN ต้องยืนยันด้วย Smart Key หรือ Fingerprint ก่อนเปิดประตู (เพิ่มความปลอดภัย)

**ขั้นตอน (ต้องลงทะเบียน PIN + Smart Key/Fingerprint ก่อน):**
1. กดปุ่ม **Register** ค้างไว้ **5 วินาที** — LED ทั้งหมดติด, หลัง 3 วินาทีเสียงเตือนหยุด, หลัง 2 วินาทีติดอีกครั้ง
2. ใส่รหัส PIN ปัจจุบัน แล้วกด `[*]` — LED แผงหน้ากระพริบ
3. กด **[1]** เพื่อปิด Dual-Mode Security หรือ **[3]** เพื่อเปิด
4. เมื่อได้ยินเสียงยืนยัน เสร็จสิ้น

**วิธีเปิดประตูเมื่อ Dual-Mode เปิดอยู่:**
- **PIN + Fingerprint**, หรือ
- **PIN + Smart Card**

:::note หมายเหตุสำคัญ
- เมื่ออยู่ใน Dual-Mode Security Entry ปุ่มอื่นจะใช้ไม่ได้
- ต้องกด Smart Key/Fingerprint ภายใน 10 วินาที ไม่งั้นจะหมดเวลา
- Guest PIN ไม่ต้องการการยืนยัน 2 ชั้น — ใช้ได้เลย
- ถ้า Guest PIN ตรงกับ User PIN → ไม่ต้องยืนยัน 2 ชั้น
- การยืนยัน 2 ชั้นใช้กฎเดียวกับ Successful/Failed Authentication และ 1-minute Lock
:::

## 7. โมดูล Home Network และรีโมทคอนโทรล (Home Network Module & Remote Control)

เปิดประตูจากระยะไกลผ่านอุปกรณ์ Home Network หรือรีโมทคอนโทรล (อุปกรณ์เสริม)

**การลงทะเบียน:**
1. กดปุ่ม **Registration** — เสียงบี๊บ
2. ใส่รหัส PIN แล้วกด `[*]`
3. ตรวจสอบว่าตัวเลข [1, 2, 3, 4, 8, 9] ติดบนคีย์แพด
4. สำหรับ **Home Network Module** กดปุ่ม **Registration** / สำหรับ **Remote Control** กดปุ่ม **Open**
5. กด `[*]` — ได้ยินเพลง เสร็จสิ้น

:::note
- ลงทะเบียนรีโมทเพิ่มได้โดยทำซ้ำขั้นตอนที่ 4
- **สูงสุด 5 รีโมท**
:::

**การลบ:**
1. กดปุ่ม **Registration** 3 วินาที
2. ใส่รหัส PIN แล้วกด `[*]`
3. ตรวจสอบตัวเลข [1, 2, 3, 4, 8, 9] ติด
4. กด **[0]**
5. กด `[*]` ค้างไว้ **5 วินาที** — ได้ยินเพลง เสร็จสิ้น

:::note
แม้ขั้นตอนที่ 3 ตัวเลข [0] ไม่ติด ก็ยังกดได้ การลงทะเบียนรีโมทใหม่จะลบรีโมทเก่าทั้งหมด (วิธีสำรองสำหรับลบรีโมท)
:::

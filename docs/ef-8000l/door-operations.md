# ขั้นตอนการเปิด/ปิดประตู

## การเปิดประตูจากด้านนอก

### เปิดด้วยรหัส PIN

1. สัมผัสคีย์แพดตัวเลข
2. ป้อนรหัส PIN แล้วตามด้วยปุ่ม [✱]
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

### เปิดด้วยฟังก์ชันสุ่มตัวเลข

ป้อนรหัสสุ่มใดๆ + ป้อนรหัส PIN + ปุ่ม [✱] หรือ ป้อนรหัส PIN + ป้อนรหัสสุ่มใดๆ + ปุ่ม [✱] ไม่ว่าจะวิธีใด เมื่อเสียงเพลงเมโลดี้ดังขึ้น ประตูจะเปิดออก

:::caution[ข้อควรระวัง]
เมื่อเปิดระบบสัมผัสหลายจุด (Multi-touch) หลังจากป้อนรหัส PIN แล้ว ตัวเลขสุ่ม 3 หลักจะแสดงบนหน้าจอ ให้กดตัวเลขทั้ง 3 หลักตามด้วยปุ่ม [✱] ประตูจึงจะเปิด เมื่อเปิดระบบ Multi-touch ฟังก์ชันรหัสสุ่มปกติจะถูกปิดใช้งาน โดยฟังก์ชันรหัสสุ่มปกติจะถูกเปิดใช้งานเป็นค่าเริ่มต้น
:::

:::note[หมายเหตุ]
ฟังก์ชันรหัสสุ่มคือการสุ่มชุดตัวเลขเพื่อป้อนก่อนหรือหลังรหัส PIN จริง ใช้เพื่อป้องกันการล่วงรู้รหัสผ่าน โดยจำนวนหลักทั้งหมดรวมรหัสผ่านจริงแล้วต้องไม่เกิน 20 หลัก
:::

### Opening by Smart Key

1. ในโหมดตรวจจับอัตโนมัติ: ทาบบัตรสมาร์ทคีย์ที่ลงทะเบียนไว้บนเครื่องอ่าน เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก
2. ในโหมดตรวจจับด้วยตนเอง: สัมผัสคีย์แพดตัวเลขเพื่อให้หน้าจอสว่างขึ้น จากนั้นทาบบัตรสมาร์ทคีย์ที่ลงทะเบียนไว้บนตัวเครื่องอ่านสัญญาณ เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก

:::note[หมายเหตุ]
สามารถใช้การ์ดที่มาพร้อมชุดเครื่อง รวมถึงบัตรโดยสาร บัตรเครดิต หรือสมาร์ทโฟนที่รองรับร่วมกันได้ โดยแตะคีย์แพดตัวเลขให้หน้าจอสว่าง แล้วทาบการ์ดบนเซ็นเซอร์ ประตูจะเปิดออก (เฉพาะการ์ดและสมาร์ทโฟนรุ่นที่เข้ากันได้เท่านั้น)
:::

### เปิดด้วยลายนิ้วมือ

1. แตะแถบเซ็นเซอร์ทำงาน
2. แตะนิ้วมือที่ลงทะเบียนไว้บนเครื่องอ่านลายนิ้วมือ
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

:::note[หมายเหตุ]
การสแกนลายนิ้วมือผิดพลาดติดต่อกัน 5 ครั้ง จะปิดล็อกเครื่องเป็นเวลา 1 นาที โดยสามารถเปิดฝาครอบสแกนได้โดยการผลักฝาครอบหรือสัมผัส [Function Sensor Bar]
:::

## การปิดประตูจากด้านนอก

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติ 2 วินาทีหลังจากประตูปิดลง (โปรดดูคู่มือประกอบเพิ่มเติม)

### โหมดล็อกด้วยตนเอง

สัมผัสคีย์แพดตัวเลขเพื่อล็อกประตู (โปรดดูรายละเอียดคู่มือเพิ่มเติม)

## การเปิดประตูจากด้านใน

### เปิดด้วยมือจับประตูด้านใน

ในขณะที่ประตูปิดอยู่ ให้กดที่จับลงเพื่อเปิดประตู

## การปิดประตูจากด้านใน

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

ในโหมดล็อกด้วยตนเอง ประตูจะไม่ล็อกโดยอัตโนมัติเมื่อปิด

1. ปิดประตูให้สนิท
2. กดปุ่ม [Open/Close] หรือหมุนปุ่มบิดล็อกด้านในเพื่อทำการล็อกประตู

# ประวัติ (History)

เมนู **History** แสดงประวัติการใช้งานทั้งหมดของล็อกที่คุณจัดการ

## 📋 ประเภทข้อมูลในประวัติ

| ประเภท | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| **Entry** | การเปิดประตูสำเร็จ | "พ่อ opened door at 09:30 via PIN" |
| **Failed attempt** | การยืนยันตัวตนล้มเหลว | "Unknown tried PIN at 22:15 (failed)" |
| **Guest entry** | Guest Key ใช้เปิดประตู | "Guest opened door at 14:00" |
| **Tamper/Alarm** | เหตุการณ์ผิดปกติ | "Trick alarm triggered at 03:00" |
| **Settings change** | การเปลี่ยนแปลงการตั้งค่า | "Super Admin changed PIN at 10:00" |

## 🔍 ดูประวัติ

**ขั้นตอน:**
1. เข้าเมนู **History** จากหน้าจอหลัก
2. เลือก **ล็อก** ที่ต้องการดู (ถ้ามีหลายล็อก)
3. เลือก **ช่วงเวลา**:
   - **Today** (วันนี้)
   - **This week** (สัปดาห์นี้)
   - **This month** (เดือนนี้)
   - **Custom** (กำหนดเอง)
4. ดูรายการเหตุการณ์

## 📊 ข้อมูลในแต่ละรายการ

แต่ละรายการในประวัติแสดง:

- **วัน-เวลา** (Date & Time)
- **ชื่อผู้ใช้** (ถ้ามี) หรือ "Unknown"
- **วิธียืนยันตัวตน** (PIN, Card, Fingerprint, App, Guest Key)
- **ล็อก** (Door Lock) — ถ้ามีหลายล็อก
- **สถานะ** (Success / Failed / Alarm)
- **หมายเหตุ** (เช่น "Trick alarm", "Low battery")

## 📤 Export ประวัติ

**ขั้นตอน:**
1. ในหน้า History → เลือกล็อก + ช่วงเวลา
2. กด **Export** หรือ **Share**
3. เลือกรูปแบบ:
   - **CSV** — เปิดใน Excel/Sheets
   - **PDF** — เก็บเป็น PDF
   - **Share** — ส่งทาง email/messenger

## 🔍 กรองข้อมูล (Filter)

สามารถกรองประวัติตาม:

- **ประเภทเหตุการณ์**: Entry / Failed / Alarm / Settings
- **ผู้ใช้**: คนใดคนหนึ่ง, Guest, Unknown
- **วิธียืนยัน**: PIN, Card, Fingerprint, App, Guest Key
- **ช่วงเวลา**: วัน/สัปดาห์/เดือน/กำหนดเอง
- **ล็อก**: ล็อกใดล็อกหนึ่ง หรือทั้งหมด

## 💡 Tips การใช้ History

### ใช้ตรวจสอบความปลอดภัย

- **ตรวจ Failed attempts** — ถ้ามีคนลอง PIN/บัตรผิดบ่อย อาจมีคนพยายมเข้า
- **ตรวจ Unknown entries** — ถ้ามี "Unknown" เข้าบ่อย อาจมีคนที่ไม่ได้ลงทะเบียน
- **ตรวจ Alarm events** — Trick alarm, High temp, Low battery
- **ตรวจ Guest entries** — ติดตามว่า Guest Key ถูกใช้อย่างไร

### ใช้เพื่อการบริหาร

- **ดูสถิติการใช้งาน** — ล็อกไหนถูกใช้บ่อย
- **ตรวจสอบการเข้าออกของพนักงาน** — ใช้กับ Wi-Fi Bridge + Member
- **ตรวจสอบ Guest Key** — ใครมาเยือนเมื่อไหร่
- **Audit trail** — สำหรับการตรวจสอบย้อนหลัง

## ⚠️ ข้อควรระวัง

- **ประวัติอาจถูกลบ** หากลบล็อกออกจากแอป (ขึ้นกับการตั้งค่า)
- **ประวัติเก็บตามเวลาของเซิร์ฟเวอร์** (ไม่ใช่เวลาท้องถิ่น)
- **ความแม่นยำของเวลา** ขึ้นกับการตั้งค่า Wi-Fi Bridge
- **ข้อมูล Failed attempts** เก็บไว้ 90 วัน (อาจแตกต่าง�ามเวอร์ชัน)
- **Guest entries** แสดงรายละเอียดน้อยกว่า Member entries

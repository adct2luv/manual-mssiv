# การจัดการล็อก (Door Lock Management)

เมนู **Door Lock** จากหน้าจอหลัก ใช้เชื่อมต่อและจัดการกุญแจ EPIC ทั้งหมด

## 🔌 เชื่อมต่อล็อก (Add Lock)

ต้อง Login ด้วย **Super Admin** ก่อน

### เชื่อมต่อผ่าน Bluetooth (BLE)

:::tip วิธีที่ง่ายที่สุด
Bluetooth เป็นวิธีเริ่มต้นที่ดีที่สุดสำหรับการตั้งค่าครั้งแรก
:::

**ขั้นตอน:**
1. เปิด **Bluetooth** บนสมาร์ทโฟน
2. ในแอป กด **+ Add** ที่เมนู Door Lock
3. เลือก **Bluetooth** mode
4. ที่ตัวล็อก กดปุ่ม **Registration (R)**
5. ใส่ **Super Admin PIN** แล้วกด `[#]`
6. แอปจะค้นหาล็อกในระยะใกล้ (ชื่อ EPIC xxx BLE)
7. เลือกล็อกที่ต้องการ
8. กด **Add** → ได้ยินเสียงยืนยัน

### เชื่อมต่อผ่าน Wi-Fi Bridge (ระยะไกล)

:::note ต้องเพิ่ม Wi-Fi Bridge ก่อน
ต้องตั้งค่า Wi-Fi Bridge ในเมนู Wifi Bridge ก่อน
:::

**ขั้นตอน:**
1. กด **+ Add** → เลือก **Wi-Fi**
2. เลือก **Wi-Fi Bridge** ที่ลงทะเบียนไว้
3. ที่ตัวล็อก กดปุ่ม **Registration (R)**
4. ใส่ **Super Admin PIN** + `[#]`
5. แอปจะเชื่อมต่อผ่าน Bridge

### หมายเหตุสำคัญ

:::warning
- **ต้องตั้ง Super Admin PIN ก่อน** เชื่อมต่อล็อก
- **ล็อก 1 เครื่อง** ต่อ **Super Admin 1 คน** เท่านั้น
- ถ้ามีคนอื่นเชื่อมต่อก่อน ต้อง **ลบล็อก** ออกจากบัญชีนั้นก่อน
- **Guest Key ใช้ได้เฉพาะ Wi-Fi mode** (ไม่ใช่ Bluetooth)
:::

## 📋 รายการล็อก (Lock List)

หน้า Door Lock แสดง:

- **ชื่อล็อก** (ตั้งโดย Super Admin)
- **รุ่น** (เช่น ES-K70, EF-P8800K)
- **Serial Number** (EBR...)
- **สถานะ** (Online/Offline)
- **โหมดการเชื่อมต่อ** (Bluetooth/Wi-Fi)
- **แบตเตอรี่** (ระดับ)

ล็อกที่คุณเพิ่มจะอยู่ในรายการ สามารถแตะเพื่อ:
- เปิด/ปิดประตู
- ดูสถานะ
- ตั้งค่า (Key, Member)
- ดูประวัติ

## ✏️ แก้ไขข้อมูลล็อก

**ขั้นตอน:**
1. เลือกล็อกจากรายการ
2. กด **Edit** (หรือไอคอนดินสอ)
3. แก้ไข:
   - **Name** (ชื่อล็อก เช่น "ประตูหน้า", "ห้องสำนักงาน")
4. กด **Save**

## 🗑 ลบล็อก (Remove Lock)

**ขั้นตอน:**
1. เลือกล็อกจากรายการ
2. กด **Delete** (หรือไอคอนถังขยะ)
3. ยืนยันการลบ

:::caution
- การลบล็อกจากแอป **ไม่ลบล็อกออกจากตัวล็อกจริง**
- ล็อกยังคงอยู่ — เพียงแต่ตัดการเชื่อมต่อกับแอป
- **กุญแจทั้งหมด (PIN, Card, Fingerprint) ยังอยู่**
- สามารถเชื่อมต่อใหม่ได้ภายหลัง
:::

## 🔄 เปลี่ยนโหมดการเชื่อมต่อ (Switch Mode)

**Bluetooth ↔ Wi-Fi:**

:::info
ล็อกสามารถสลับระหว่าง Bluetooth และ Wi-Fi ได้
- Bluetooth ใช้ได้ในระยะใกล้ (≤ 10 เมตร)
- Wi-Fi ใช้ได้ทุกที่ (ต้องมี Wi-Fi Bridge)
- **Wi-Fi mode รองรับ Guest Key** และควบคุมระยะไกล
:::

**ขั้นตอน:**
1. เปิดเมนู **Door Lock** เลือกล็อก
2. กด **Connection Mode**
3. เลือก **Bluetooth** หรือ **Wi-Fi**

## 📋 ตัวอย่างการใช้งาน

### ตั้งค่าครั้งแรก
1. เปิด Bluetooth บนสมาร์ทโฟน
2. Login เป็น Super Admin
3. เชื่อมต่อล็อกผ่าน Bluetooth
4. ตั้งชื่อล็อก (เช่น "ประตูหน้า")
5. เพิ่มสมาชิก (General Admin / User)
6. เพิ่มกุญแจ (PIN, Card, Fingerprint)
7. ตั้งค่า Wi-Fi Bridge (ถ้าต้องการ)

### ใช้งานระยะไกล
1. สลับโหมดเป็น **Wi-Fi**
2. เปิด/ปิดประตูจากทุกที่
3. สร้าง Guest Key ให้แขก
4. ดูประวัติการเข้า-ออก

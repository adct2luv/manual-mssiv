# การติดตั้งและตั้งค่าเริ่มต้น

## 1. ดาวน์โหลดแอป

แอป EPIC Things มีให้ดาวน์โหลดฟรีทั้ง iOS และ Android:

- **iOS:** App Store → ค้นหา "EPIC Things"
- **Android:** Google Play Store → ค้นหา "EPIC Things"

## 2. สร้างบัญชีผู้ใช้ (Sign Up)

1. เปิดแอป EPIC Things
2. คลิก **Sign Up** หรือ **สร้างบัญชีใหม่**
3. กรอกข้อมูล:
   - อีเมล
   - รหัสผ่าน (แนะนำ 8 ตัวอักษรขึ้นไป ผสมตัวเลขและตัวอักษรพิเศษ)
   - ยืนยันรหัสผ่าน
4. ยอมรับข้อกำหนดการใช้งาน
5. ยืนยันอีเมล (จะมีลิงก์ยืนยันส่งไป)

## 3. เข้าสู่ระบบ (Login)

1. เปิดแอป
2. กรอกอีเมลและรหัสผ่าน
3. คลิก **Login** หรือ **เข้าสู่ระบบ**

:::tip[Guest Login]
หากต้องการทดลองใช้แอปก่อนสมัคร ใช้ **Guest Login** ได้ แต่ฟีเจอร์บางอย่างจะถูกจำกัด
:::

## 4. ตั้งค่า Super Admin PIN

หลังสมัครและล็อกอินครั้งแรก ระบบจะให้ตั้ง **Super Admin PIN**:

1. ใส่รหัส PIN 4–12 หลัก
2. ยืนยันรหัส PIN อีกครั้ง
3. **สำคัญ:** จดรหัสนี้ไว้ในที่ปลอดภัย หากลืมต้องติดต่อช่างเท่านั้น

## 5. เชื่อมต่อกุญแจ (Door Lock Registration)

### 5.1 ผ่าน Bluetooth (BLE)

1. เปิด Bluetooth บนสมาร์ทโฟน
2. ในแอป คลิก **+ Add Lock** หรือ **เพิ่มกุญแจ**
3. เลือกโหมด **Bluetooth**
4. บนตัวล็อก กดปุ่ม Registration
5. แอปจะค้นหาล็อกในระยะใกล้
6. เลือกล็อกที่ต้องการเชื่อมต่อ
7. ใส่ Super Admin PIN
8. ตั้งชื่อล็อก เช่น "ประตูหน้า"

### 5.2 ผ่าน Wi-Fi Bridge (ระยะไกล)

:::note[ต้องมี Wi-Fi Bridge]
การควบคุมระยะไกลต้องใช้ **Wi-Fi Bridge** (อุปกรณ์เสริม ซื้อแยก)
:::

1. เสียบ Wi-Fi Bridge เข้ากับปลั๊กไฟใกล้กับกุญแจ
2. ในแอป คลิก **+ Add Bridge**
3. เลือก Wi-Fi ของบ้าน
4. ใส่รหัสผ่าน Wi-Fi
5. กดปุ่ม pairing บน Bridge
6. แอปจะค้นหาและเชื่อมต่อ Bridge
7. หลังเชื่อม Bridge แล้ว สามารถเพิ่มล็อกผ่าน Bridge ได้

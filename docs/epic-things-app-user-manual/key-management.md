# การจัดการกุญแจและสมาชิก

## 🔑 การจัดการกุญแจ (Key Management)

หลังเชื่อมต่อล็อกแล้ว สามารถจัดการกุญแจได้หลายประเภทผ่านแอป:

### PIN (รหัสผ่าน)

1. เลือกล็อกจากหน้า Home
2. ไปที่ **Key Management** → **PIN**
3. คลิก **+ Add PIN**
4. ใส่รหัส PIN 4–12 หลัก
5. ตั้งชื่อผู้ใช้ (เช่น "พ่อ", "แม่", "ลูก")
6. ตั้งวันหมดอายุ (ถ้าต้องการ)
7. คลิก **Save**

### Card (บัตร RFID)

1. ไปที่ **Key Management** → **Card**
2. คลิก **+ Add Card**
3. ใส่ชื่อผู้ใช้
4. แตะบัตรที่เครื่องอ่านบนตัวล็อก
5. แอปจะยืนยันว่าบัตรถูกเพิ่มแล้ว
6. คลิก **Save**

### Fingerprint (ลายนิ้วมือ)

1. ไปที่ **Key Management** → **Fingerprint**
2. คลิก **+ Add Fingerprint**
3. ใส่ชื่อผู้ใช้
4. บนตัวล็อก วางนิ้วบนเซ็นเซอร์ 3 ครั้งตามคำแนะนำ
5. แอปจะแสดงผลการลงทะเบียน
6. คลิก **Save**

### Remote Control (รีโมท)

1. ไปที่ **Key Management** → **Remote**
2. คลิก **+ Add Remote**
3. กดปุ่ม pairing บนรีโมท
4. ตั้งชื่อรีโมท
5. คลิก **Save**

### Face ID (จดจำใบหน้า — เฉพาะรุ่นที่รองรับ)

1. ไปที่ **Key Management** → **Face**
2. คลิก **+ Add Face**
3. ยืนยันตัวตนด้วย Super Admin PIN
4. บนตัวล็อก มองเข้าเซ็นเซอร์กล้อง
5. รอให้ระบบจดจำใบห้าเสร็จ
6. คลิก **Save**

## 👥 การจัดการสมาชิก (Member Management)

### เพิ่มสมาชิก

1. ไปที่ **Members** → **+ Add Member**
2. กรอกข้อมูล:
   - ชื่อ-นามสกุล
   - อีเมล
   - เบอร์โทรศัพท์ (ไม่บังคับ)
   - ประเภท: **Admin** / **Member** / **Guest**
3. กำหนดล็อกที่สมาชิกเข้าถึงได้
4. คลิก **Invite** หรือ **Send** — ระบบจะส่งคำเชิญทางอีเมล

### ประเภทสมาชิก

| ประเภท | สิทธิ์ |
|---|---|
| **Admin** | จัดการล็อกทั้งหมด + เพิ่ม/ลบสมาชิก + ดูประวัติ |
| **Member** | ใช้กุญแจ + ดูประวัติของตัวเอง |
| **Guest** | ใช้กุญแจชั่วคราว (มีวันหมดอายุ) |

### ลบสมาชิก

1. ไปที่ **Members**
2. เลือกสมาชิกที่ต้องการลบ
3. คลิก **Delete Member** หรือ **Remove**
4. ยืนยันการลบ

:::warning
การลบสมาชิกจะ **ลบการเข้าถึงล็อกทันที** แต่จะไม่ลบกุญแจ (PIN/บัตร/ลายนิ้วมือ) ที่ลงทะเบียนไว้
:::

## 📜 การจัดการประวัติ (History Management)

1. ไปที่ **History**
2. เลือกล็อกที่ต้องการดู
3. เลือกช่วงเวลา (วันนี้ / สัปดาห์นี้ / เดือนนี้ / กำหนดเอง)
4. ดูรายการ:
   - วัน-เวลา
   - ชื่อผู้ใช้ (ถ้ามี)
   - วิธีเปิด (PIN, Card, Fingerprint, App)
   - สถานะ (สำเร็จ / ล้มเหลว)
5. สามารถ **Export CSV** หรือ **แชร์** ผ่านแอปอื่นได้

# การจัดการกุญแจ (Key Management)

เมนู **Key** ใช้จัดการวิธียืนยันตัวตนที่ลงทะเบียนกับล็อก

:::tip ลงทะเบียนได้เร็วผ่านแอป
คุณสามารถลงทะเบียนวิธียืนยันตัวตนผ่านแอปได้อย่างรวดเร็ว
**แต่ PIN ไม่สามารถลงทะเบียนผ่านแอปโดยตรง** — ต้องทำที่ตัวล็อก
:::

## 📋 ประเภทกุญแจที่รองรับ

| ประเภท | คำอธิบาย | ลงทะเบียนผ่านแอป | จำกัด |
|---|---|---|---|
| **PIN** | รหัสตัวเลข 4-12 หลัก | ❌ ต้องทำที่ตัวล็อก | 1 PIN ต่อล็อก |
| **Card** | Smart Card / RFID 13.56 MHz | ✅ | สูงสุด 200 ใบ |
| **Fingerprint** | ลายนิ้วมือ | ✅ | สูงสุด 100 ลาย |
| **Remote** | รีโมทคอนโทรล | ✅ | ตามรุ่น |
| **Face ID** | จดจำใบหน้า | ✅ | สูงสุด 30 ใบ |

## 1. การจัดการ PIN

### ดู PIN ปัจจุบัน

**ขั้นตอน:**
1. ไปที่ล็อก → กด **Key** → แท็บ **PIN**
2. เลือก **User PIN #1** เพื่อดู PIN ปัจจุบัน
3. กด **Change PIN** เพื่อเปลี่ยน

:::info
- PIN ที่ลงทะเบียนในแอปคือ **User PIN** ไม่ใช่ Super Admin PIN
- PIN ต้อง **≥ 4 หลัก**
- ตั้ง **User PIN ได้แค่ 1 ชุด** ต่อล็อก
:::

### เปลี่ยน PIN ผ่านแอป

**ขั้นตอน:**
1. **PIN** tab → กด **Change PIN**
2. ใส่ **PIN ใหม่** (≥ 4 หลัก)
3. กด **Confirm** → สำเร็จ

:::warning
ต้องเปลี่ยน PIN ที่ตัวล็อกก่อน (ถ้า PIN ตั้งเดิมคือ 1234) แล้วจึงเปลี่ยนในแอปได้
- ถ้ายังไม่เคยเปลี่ยน → ต้องไปที่ตัวล็อกก่อน
:::

## 2. การจัดการ Card (Smart Card)

### ดูรายการ Card

**PIN** tab ถัดไป → **Card** tab

แสดง:
- **Card #1, #2, ...** (หมายเลขบัตร)
- สถานะ (ลงทะเบียนแล้ว/ว่าง)
- ชื่อผู้ใช้ (ถ้ามี)

### เพิ่ม Card ใหม่

**ขั้นตอน:**
1. แท็บ **Card** → กด **+ Add**
2. **แตะบัตร Smart Card** ที่เครื่องอ่านของตัวล็อก
3. แอปจะจดจำบัตรและเพิ่มเข้าระบบ
4. กด **Save** หรือ **Close**

:::tip
- บัตร Smart Card ต้องเป็น **13.56 MHz** (Mifare / FeliCa)
- รองรับสูงสุด **200 ใบ**
- บัตรที่อยู่ในกล่องตอนซื้อล็อกสามารถใช้ได้ทันที
:::

### ลบ Card

**ขั้นตอน:**
1. เลือกบัตรที่ต้องการลบ
2. กด **Delete** (ไอคอนถังขยะ)
3. ยืนยันการลบ

## 3. การจัดการ Fingerprint

### เพิ่มลายนิ้วมือ

**ขั้นตอน:**
1. แท็บ **Fingerprint** → กด **+ Add**
2. **วางนิ้วบนเซ็นเซอร์ 4 ครั้ง** (ที่ตัวล็อก)
3. แต่ละครั้งจะมีเสียงบี๊บ + ไฟ LED แสดงความคืบหน้า
4. เมื่อสำเร็จ → แอปแสดงผลสำเร็จ + บันทึก

:::info
- ต้องวางนิ้ว **4 ครั้ง** ในตำแหน่งเดียวกัน
- รองรับสูงสุด **100 ลายนิ้วมือ**
- ลายนิ้วมือเดียวต่อผู้ใช้
:::

### ลบลายนิ้วมือ

**ขั้นตอน:**
1. เลือกลายนิ้วมือที่ต้องการ
2. กด **Delete**
3. ยืนยัน

## 4. การจัดการ Remote Control

**ขั้นตอน:**
1. แท็บ **Remote** → กด **+ Add**
2. **กดปุ่ม pairing บนรีโมท** (จนไฟกระพริบ)
3. แอปจะจับคู่กับรีโมท
4. ตั้งชื่อรีโมท (เช่น "รีโมตห้องนั่งเล่น")
5. กด **Save**

### ลบ Remote

**ขั้นตอน:**
1. เลือกรีโมท → กด **Delete**
2. ยืนยัน

## 5. การจัดการ Face ID (รุ่นที่รองรับ)

**ขั้นตอน:**
1. แท็บ **Face** → กด **+ Add**
2. **มองตรงเข้ากล้อง** (บนตัวล็อก) ในระยะ **1 เมตร**
3. รอให้ระบบจดจำ (LED เขียว = สำเร็จ)
4. ตั้งชื่อ (เช่น "พ่อ", "แม่")
5. กด **Save**

### ลบ Face ID

**ขั้นตอน:**
1. เลือก Face → กด **Delete**
2. ยืนยัน

## 🔄 เปลี่ยนชื่อผู้ใช้ (Change User Name)

**ขั้นตอน:**
1. เลือกกุญแจ (PIN, Card, Fingerprint, Remote, Face) ที่ต้องการ
2. กด **Change User** หรือไอคอนดินสอ
3. ใส่ชื่อใหม่ (เช่น "พ่อ", "ลูกสาว")
4. กด **Confirm** → ชื่อจะปรากฏในประวัติ

## 📊 ดูสถิติกุญแจ

- **Total count** — จำนวนกุญแจทั้งหมดที่ลงทะเบียน
- **Available** — จำนวนที่ยังลงทะเบียนได้
- **By type** — จำนวนแยกตามประเภท (PIN/Card/FP/Remote/Face)

## ⚠️ ข้อควรระวัง

- **PIN ไม่สามารถลงทะเบียนผ่านแอปได้** — ต้องทำที่ตัวล็อก
- **กุญแจที่ลบแล้วไม่สามารถกู้คืนได้** — ต้องลงทะเบียนใหม่
- **Card 1 ใบ** ต่อผู้ใช้ 1 คน (ถ้ามีหลายคนต้องหลายบัตร)
- **ลายนิ้วมือที่ชำรุด/เปียก** อาจสแกนไม่ติด → ใช้ PIN แทน
- **5 ครั้งที่ยืนยันผิด** = Anti-Prank + ล็อก 1 นาที
- **ลบกุญแจแล้ว User ใน Member ยังอยู่** — ต้องลบ Member แยกต่างหาก

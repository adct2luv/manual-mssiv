# EPIC Things APP — คู่มือผู้ใช้

แอป **EPIC Things** เป็นแอปมือถือที่ให้คุณ **ควบคุมและจัดการกุญแจดิจิทัล EPIC** ผ่านสมาร์ทโฟน รองรับทั้ง iOS และ Android ใช้การเชื่อมต่อผ่าน **Bluetooth (BLE)** และ **Wi-Fi Bridge** สำหรับการควบคุมจากระยะไกล

## 📱 ฟีเจอร์หลัก

- **Super Admin PIN** — รหัสผ่านหลักสำหรับผู้ดูแลระบบ
- **Guest Login** — ล็อกอินสำหรับผู้ใช้ชั่วคราว
- **Member Management** — จัดการสมาชิกและสิทธิ์การเข้าถึง
- **Door Lock Registration** — ลงทะเบียนกุญแจดิจิทัลเข้ากับแอป
- **WiFi Bridge Management** — จัดการการเชื่อมต่อ Wi-Fi
- **Key Management** — จัดการ PIN, Card, Fingerprint, Remote Control, Face ID
- **History Management** — ดูประวัติการเข้า-ออก

## ⚠️ คำเตือน

- **อย่าใช้แบตเตอรี่แบบชาร์จไฟได้** ในกุญแจที่จัดการผ่านแอป
- **อย่าให้น้ำโดนตัวล็อก** โดยตรง
- **เก็บรหัสผ่าน Wi-Fi ไว้เป็นความลับ** เพราะทุกคนที่เชื่อมต่อ Wi-Fi เดียวกันสามารถควบคุมล็อกได้

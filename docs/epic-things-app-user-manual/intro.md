# EPIC Things App — Quick Guide

แอป **EPIC Things** ควบคุมและจัดการกุญแจดิจิทัล EPIC ผ่านสมาร์ทโฟน รองรับ **Bluetooth (BLE)** และ **Wi-Fi Bridge** สำหรับควบคุมระยะไกล

## 📱 ดาวน์โหลดแอป

- **iOS:** App Store → ค้นหา "EPIC Things"
- **Android:** Google Play Store → ค้นหา "EPIC Things"

QR Code สำหรับดาวน์โหลดมีอยู่ในหน้า 3 ของ PDF

## 🎯 ต้องมีอะไรบ้าง

:::tip Super Admin PIN จำเป็น
ในการใช้งาน EPIC Things App ต้องมี **Super Admin PIN** ก่อน

**Super Admin คืออะไร:**
EPIC Door Lock มี 3 ระดับผู้ใช้:
- **Super Admin** (ระดับสูงสุด) — เชื่อมต่อแอปกับล็อก จัดการล็อก
- **General Admin**
- **User** (ระดับผู้ใช้ทั่วไป)

**สิทธิ์ของ Super Admin:**
- ควบคุมการเข้าถึงฟังก์ชันทั้งหมดของล็อก
- เชื่อมต่อแอปกับล็อกประตู (ป้องกันผู้อื่นเชื่อมต่อโดยไม่ได้รับอนุญาต)
- เพิ่ม/ลบสมาชิก, ตั้งค่าต่างๆ
:::

## ✨ ฟีเจอร์หลัก

- **ควบคุมล็อกผ่าน Bluetooth (BLE)** — ใช้งานใกล้ๆ
- **ควบคุมล็อกผ่าน Wi-Fi Bridge** — ใช้งานระยะไกล
- **จัดการสมาชิก** — เพิ่ม/ลบ ผู้ใช้, กำหนดสิทธิ์
- **จัดการกุญแจ** — PIN, Card, Fingerprint, Remote, Face ID
- **ดูประวัติ** — การเข้า-ออกทั้งหมด
- **ตั้งค่า Guest Key** — ให้แขกผู้มาเยือนใช้ชั่วคราว
- **Bluetooth + Wi-Fi** — สลับโหมดการเชื่อมต่อ

:::warning ข้อควรระวัง
- **Super Admin PIN ต้องตั้งก่อน** ใช้งานแอป
- **เก็บ Super Admin PIN ไว้ในที่ปลอดภัย** — ถ้าลืมต้องติดต่อช่าง
- **Bluetooth ใช้ได้ในระยะใกล้** — Wi-Fi Bridge ต้องใช้สำหรับ Guest Key (ระยะไกล)
- **ไม่สามารถลงทะเบียน PIN ผ่านแอปโดยตรง** — ต้องทำที่ตัวล็อก
:::

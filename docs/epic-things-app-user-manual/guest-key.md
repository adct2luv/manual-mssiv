# กุญแจแขก (Guest Key)

Guest Key เป็นฟีเจอร์พิเศษที่ให้ผู้มาเยือน (เช่น แขก, ช่างซ่อม) เปิดประตูได้ **ครั้งเดียว** โดยไม่ต้องลงทะเบียน Card/Fingerprint เพิ่ม

## 🎯 ข้อกำหนด

:::warning ต้องใช้โหมด Wi-Fi
**Guest Key ใช้ได้เฉพาะโหมด Wi-Fi** ไม่สามารถใช้กับ Bluetooth ได้
- ต้องมี **Wi-Fi Bridge** ที่ลงทะเบียนแล้ว
- ล็อกต้องอยู่ใน **โหมด Wi-Fi** (ไม่ใช่ Bluetooth)
:::

## 📋 ประเภท Guest Key

| ประเภท | ลักษณะ | ใช้กับ |
|---|---|---|
| **One-time** | ใช้ได้ 1 ครั้ง แล้วหมดอายุ | แขกมาเยือน, ช่างซ่อม |
| **Time-limited** | ใช้ได้หลายครั้งภายในเวลาที่กำหนด | ผู้เช่าระยะสั้น, พนักงานทำความสะอาด |

## ➕ สร้าง Guest Key

**ขั้นตอน:**
1. เข้าเมนู **Guest Key** จากหน้าจอหลัก
2. กด **+ Add** หรือไอคอน "+"
3. เลือกประเภท:
   - **One-time** — ใช้ได้ 1 ครั้ง
   - **Time-limited** — ตั้งเวลาหมดอายุ
4. กรอกข้อมูล:
   - **Name** (เช่น "ช่างซ่อม", "แขกวันอาทิตย์")
   - **Valid Period** (สำหรับ Time-limited)
   - **Allowed Locks** (เลือกล็อก)
5. กด **Generate** หรือ **Create**
6. **แชร์ Guest Key** ผ่าน email, SMS, QR code, หรือ copy link

## 🎫 วิธีใช้ Guest Key

### สำหรับผู้รับ Guest Key
1. เปิด **Guest Key** (ได้รับทาง email/QR/link)
2. เปิดแอป EPIC Things (หรือ Guest Web App)
3. กด **"Open Door"** ที่ปุ่มใน Guest Key
4. ล็อกจะเปิด **1 ครั้ง**

### ข้อจำกัด
- **Guest Key ใช้ได้ครั้งเดียว** แล้วจะหมดอายุอัตโนมัติ (สำหรับ One-time)
- **Guest Key ไม่ต้องลงทะเบียนบัญชี** — แค่มี link/code
- **Guest Key ต้องการ Wi-Fi mode** ไม่สามารถใช้ Bluetooth
- **ผู้รับ Guest Key ไม่ต้องการ app** (ใช้ Web App ได้)

## 📊 จัดการ Guest Key

### ดูรายการ Guest Key

หน้า Guest Key แสดง:
- **Name** (ชื่อ Guest Key)
- **Type** (One-time / Time-limited)
- **Status** (Active/Used/Expired)
- **Used by** (ใครใช้แล้ว)
- **Valid until** (วันหมดอายุ)

### ยกเลิก Guest Key

**ขั้นตอน:**
1. เลือก Guest Key ที่ต้องการ
2. กด **Cancel** หรือ **Revoke**
3. ยืนยันการยกเลิก

:::caution
- **Guest Key ที่ใช้แล้วแล้ว** ไม่สามารถใช้ซ้ำได้
- **Guest Key ที่ยกเลิกแล้ว** ไม่สามารถใช้ได้อีก
- ต้องสร้าง Guest Key ใหม่ถ้าต้องการ
:::

### ดูประวัติการใช้

แต่ละ Guest Key มี **log การใช้งาน**:
- **ใครใช้** (ถ้ามี account)
- **เมื่อไหร่** (วัน-เวลา)
- **ล็อกไหน** (เปิดล็อกไหน)
- **สำเร็จหรือไม่**

## 📋 ตัวอย่างการใช้

### แขกมาเยือน
1. Super Admin สร้าง **One-time Guest Key** "แขกบ้าน"
2. ตั้ง **Valid Period** 24 ชั่วโมง
3. **ส่งลิงก์** ให้แขกทาง email
4. แขกเปิดลิงก์ → กดปุ่ม "Open Door"
5. ประตูเปิด **1 ครั้ง** แล้ว Guest Key หมดอายุ

### ช่างซ่อมบำรุง
1. Super Admin สร้าง **Time-limited Guest Key** "ช่างซ่อม"
2. ตั้ง **Valid Period** 3 ชั่วโมง (09:00-12:00)
3. **ส่ง QR code** ให้ช่าง
4. ช่างสแกน QR → ได้สิทธิ์เข้า-ออกได้หลายครั้งใน 3 ชั่วโมง
5. หลัง 12:00 → Guest Key หมดอายุ

### ผู้เช่าระยะสั้น
1. Super Admin สร้าง **Time-limited Guest Key** "ผู้เช่า 1 เดือน"
2. ตั้ง **Valid Period** 30 วัน
3. ผู้เช่าสามารถเปิดประตูได้หลายครั้งใน 30 วัน
4. หลัง 30 วัน → Guest Key หมดอายุ

## 💡 Tips

- **สร้าง Guest Key ก่อนแขกมา** — ไม่ต้องรอให้แขกถึงแล้วค่อยสร้าง
- **ตั้งชื่อจำง่าย** — "ช่างประปา 5/5", "แขกเย็นนี้"
- **ใช้ One-time** สำหรับคนที่ไม่รู้จัก
- **ใช้ Time-limited** สำหรับคนที่ไว้ใจ (ช่าง, ผู้เช่า)
- **แชร์ผ่าน email/QR** สะดวกกว่า copy link
- **เช็ค Guest Key ที่หมดอายุ** เป็นประจำ
- **ยกเลิก Guest Key ทันที** ถ้าแขกยกเลิกกะทันหนัน

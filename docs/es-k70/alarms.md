# สัญญาณเตือนและความปลอดภัย

## กุญแจกลไกฉุกเฉิน

### ขั้นตอน

1. เปิดฝาครอบรูกุญแจฉุกเฉินซึ่งอยู่ส่วนบนของที่จับหน้าเครื่องด้านนอก โดยผลักดันฝาครอบไปทางด้านขวา
2. เสียบกุญแจสำรองฉุกเฉินเข้ากับช่องกุญแจ จากนั้นบิดหมุนกุญแจจากขวาไปซ้าย
3. เมื่อได้ยินเสียงสัญญาณ "ติ๊ก" และสลักล็อกได้เคลื่อนตัวออกสู่ตำแหน่งล็อกเริ่มต้นแล้ว ให้ปล่อยมือ
4. กดดึงที่จับประตูลงด้านล่าง
5. ผลักดันปิดฝาครอบรูกุญแจฉุกเฉิน

:::note[หมายเหตุ]
การเปิดประตูในกรณีที่ระบบล็อกอิเล็กทรอนิกส์ทำงานผิดปกติ
:::

# ขั้นตอนการเปิด/ปิดประตู

## การเปิดประตูจากด้านนอก

### เปิดด้วยรหัส PIN

1. สัมผัสคีย์แพดตัวเลข
2. ป้อนรหัส PIN แล้วตามด้วยปุ่ม [✱]
3. เมื่อได้ยินเสียงเพลงเมโลดี้ ประตูจะเปิดออก

### เปิดด้วยบัตร RFID

1. ในโหมดตรวจจับอัตโนมัติ: ทาบบัตร RFID ที่ลงทะเบียนไว้บนตัวเครื่องอ่านสัญญาณ เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก
2. ในโหมดตรวจจับด้วยตนเอง: สัมผัสคีย์แพดตัวเลขเพื่อให้หน้าจอสว่างขึ้น จากนั้นทาบบัตร RFID ที่ลงทะเบียนบนตัวเครื่องอ่านสัญญาณ เสียงเพลงเมโลดี้จะดังขึ้นและประตูจะเปิดออก

### เปิดด้วยฟังก์ชันสุ่มตัวเลข

1. สัมผัสหน้าจอคีย์แพด ป้อนตัวเลขสุ่มใดๆ ก่อนหรือหลังรหัส PIN จริง แล้วกดปุ่ม [*] เมื่อมีเสียงเพลงเมโลดี้ดังขึ้น ประตูจะปลดล็อกเปิดออก
2. ป้อนรหัสสุ่มใดๆ + ป้อนรหัส PIN + ปุ่ม [✱]
3. ป้อนรหัส PIN + ป้อนรหัสสุ่มใดๆ + ปุ่ม [✱]
4. ไม่ว่าจะวิธีใด เมื่อเสียงเพลงเมโลดี้ดังขึ้น ประตูจะเปิดออก

### Opening by Dual-mode Security

ป้อนรหัส PIN + ปุ่ม [✱] + ทาบบัตรที่จะลงทะเบียนบนพื้นที่เครื่องอ่านการ์ด

## การปิดประตูจากด้านนอก

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

สัมผัสคีย์แพดตัวเลขแล้วกดปุ่ม [✱]

## การเปิดประตูจากด้านใน

### เปิดด้วยมือจับประตูด้านใน

ดึงหรือกดที่จับประตูด้านในลงเพื่อปลดล็อกสลักเกลียว (Deadbolt) และเปิดประตู

## การปิดประตูจากด้านใน

### โหมดล็อกอัตโนมัติ

ประตูจะล็อกโดยอัตโนมัติภายใน 2 วินาทีหลังจากประตูปิดสนิท

### โหมดล็อกด้วยตนเอง

หลังจากประตูปิดแล้ว ให้กดปุ่ม [Open/Close] ประตูจะล็อก

:::note[หมายเหตุ]
ในโหมดล็อกด้วยตนเอง เมื่อกดปุ่ม [Manual Lock] จะได้ยินเสียงบี๊บดังขึ้นเพื่อส่งสัญญาณว่าประตูล็อกแล้ว
:::

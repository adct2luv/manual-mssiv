# ส่วนประกอบ

## ตัวล็อกด้านนอก

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **Number pad** |  |
| 2 | **Status LED lamp** |  |
| 3 | **Battery lamp** |  |
| 4 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |
| 5 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 6 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |
| 7 | **Mechanical key hole cover** |  |
| 8 | **ที่จับประตูด้านนอก** | ดึงหรือหมุนลงเพื่อเปิดประตูหลังปลดล็อกสำเร็จ |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |
| 2 | **Alkaline AA Battery (×8)** |  |
| 3 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 4 | **ปุ่มเปิด/ปิดประตู (Open/Close)** | กดปุ่มเพื่อล็อกหรือปลดล็อกประตูแบบสัมผัสเดียวจากด้านใน |
| 5 | **ปุ่มล็อกสองชั้น (Dual Lock)** | เปิดโหมดความเป็นส่วนตัว เพื่อป้องกันการเปิดประตูจากภายนอก |
| 6 | **ที่จับประตูด้านใน** | ดึงหรือหมุนเพื่อเปิดประตูจากภายในห้อง |

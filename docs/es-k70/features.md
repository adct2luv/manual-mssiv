# คุณสมบัติพิเศษ

## การลงทะเบียนรหัส PIN สำหรับผู้มาเยือน (Guest PIN)

:::note[หมายเหตุ]
※ ฟังก์ชันตัวเลขสุ่มถูกเปิดการใช้งานเป็นค่าเริ่มต้นของโรงงาน ซึ่งเป็นระบบความปลอดภัยที่ยอดเยี่ยมเพื่อป้องกันมิให้บุคคลภายนอกล่วงรู้รหัส PIN จริงของคุณ โดยคุณสามารถบิดกดตัวเลขสุ่มใดๆ ก่อนหรือหลังจากป้อนรหัส PIN จริง ทั้งนี้จำนวนหลักทั้งหมดรวมรหัสผ่านจริงแล้วต้องไม่เกิน 20 หลัก
:::

## โหมดล็อกบังคับจากด้านใน (Inside Force Lock)

ระบบล็อกสองชั้นจากด้านใน - เมื่อเปิดใช้งานจะไม่ยอมรับรหัส PIN หรือคีย์การ์ดปลดล็อกประตูจากภายนอก

### ขั้นตอน

1. หลังจากประตูปิดแล้ว
2. กดปุ่ม [Safety] ที่ตัวเครื่องด้านในค้างไว้ 3 วินาที จะมีเสียงเพลงเมโลดี้ดังขึ้นและการตั้งค่าเสร็จสิ้น

**การยกเลิก:** กดที่จับประตูด้านในลงด้านล่าง ระบบล็อกสองชั้นจะถูกยกเลิก

## การตั้งค่าระดับเสียงการทำงาน

ฟังก์ชันนี้ให้ผู้ใช้ปรับระดับเสียงการทำงานทีละระดับ (ทำขณะประตูปิด) ※ ปรับเสียงได้สูงสุด 7 ระดับ

### ขั้นตอน

1. สัมผัสคีย์แพดตัวเลข
2. ป้อนรหัส PIN ที่ลงทะเบียนไว้
3. กดปุ่ม [✱]
4. กดปุ่ม [3] หรือ [6]

:::note[หมายเหตุ]
กดปุ่ม [3] เพื่อเพิ่มเสียง กดปุ่ม [6] เพื่อลดเสียง ในโหมดปิดเสียงคีย์แพดตัวเลขจะกะพริบเมื่อกดตัวเลข ระดับเสียงถูกตั้งไว้สูงสุดเป็นค่าเริ่มต้น
:::

## การตั้งค่าปิดเสียงชั่วคราว (Sound Mute)

ไม่ว่าจะตั้งระดับเสียงไว้เท่าใด ฟังก์ชันนี้ให้คุณปิดเสียงการทำงานได้ ใช้ได้เพียงครั้งเดียวเท่านั้น เสียงจะกลับมาทำงานในการใช้งานครั้งถัดไป

### ขั้นตอน

1. สัมผัสคีย์แพดตัวเลข กดปุ่ม [✱] (ในสถานะปิดเสียง)
2. ป้อนรหัส PIN กดปุ่ม [✱] ประตูจะเปิด (คีย์แพดตัวเลขจะกะพริบ)

## การตั้งค่าโหมดล็อกอัตโนมัติ / ล็อกด้วยตนเอง

คุณสมบัติเสริมที่ช่วยให้ตั้งค่าล็อกได้ทั้งโหมดอัตโนมัติและล็อกด้วยตนเอง

### ขั้นตอน

1. กดปุ่ม [Open/Close] เมื่อสลักเดดโบลต์ยื่นออกจาก Mortise ตลับล็อก
2. สัมผัสคีย์แพดตัวเลข
3. ป้อนรหัส PIN จากนั้นกดปุ่ม [✱] 2 ครั้ง
4. กด [4] เพื่อตั้งโหมดอัตโนมัติ กด [7] เพื่อตั้งโหมดล็อกด้วยตนเอง

## การตั้งค่าระบบยืนยันตัวตนสองชั้น

:::note[หมายเหตุ]
※ ฟังก์ชันตัวเลขสุ่มถูกเปิดการใช้งานเป็นค่าเริ่มต้นของโรงงาน ซึ่งเป็นระบบความปลอดภัยที่ยอดเยี่ยมเพื่อป้องกันมิให้บุคคลภายนอกล่วงรู้รหัส PIN จริงของคุณ โดยคุณสามารถบิดกดตัวเลขสุ่มใดๆ ก่อนหรือหลังจากป้อนรหัส PIN จริง ทั้งนี้จำนวนหลักทั้งหมดรวมรหัสผ่านจริงแล้วต้องไม่เกิน 20 หลัก
:::

# สัญญาณเตือนและความปลอดภัย

## สัญญาณแจ้งเตือนการเปลี่ยนแบตเตอรี่

เมื่อระบบตรวจพบว่าแบตเตอรี่อ่อน ไฟ LED [แจ้งเตือนเปลี่ยนแบตเตอรี่] จะสว่างขึ้น และตัวเครื่องจะส่งเสียงเพลงสัญญาณเมโลดี้แจ้งเตือนผู้ใช้งาน

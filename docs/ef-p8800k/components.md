# ส่วนประกอบ

## ตัวล็อกด้านนอก

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **เครื่องสแกนลายนิ้วมือ** | แตะนิ้วมือที่ลงทะเบียนเพื่อปลดล็อก |
| 2 | **แผงคีย์แพดตัวเลข** | ป้อนรหัส PIN หรือรหัสตั้งค่าการทำงาน (1-9, *, 0, #) |
| 3 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |
| 4 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 5 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |
| 2 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 3 | **ปุ่มบิดล็อกด้วยตนเอง** | ใช้เพื่อล็อกหรือปลดล็อกประตูด้วยตนเองในโหมดล็อกแบบแมนนวล |

## อุปกรณ์ในกล่อง
- แป้นรับลิ้นล็อก
- ตลับมอร์ทิส
- แบตเตอรี่ 4 ก้อน
- ชุดสกรู (4 ตัว)
- แกนหมุนตลับมอร์ทิส
- คู่มือการติดตั้ง
- บัตร RFID

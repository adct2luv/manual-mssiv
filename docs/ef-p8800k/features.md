# คุณสมบัติพิเศษ

## Guest PIN Registration

### ขั้นตอน

1. เปิดฝาครอบแบตเตอรี่ กดปุ่ม [Registration] 1 ครั้ง และจะมีเสียงสัญญาณบี๊บดังขึ้น
2. ป้อนรหัส PIN ตามด้วยปุ่ม [#]
3. ตรวจสอบว่ามีไฟสว่างที่ตัวเลข [1, 2, 3, 4, 5] บนคีย์แพดตัวเลข จากนั้นกดปุ่ม [4]
4. เมื่อมีไฟสว่างขึ้นทุกปุ่มบนคีย์แพด ให้ป้อนรหัส PIN สำหรับผู้มาเยือนจำนวน 4~12 หลัก จากนั้นกดปุ่ม [#]
5. ป้อนรหัส PIN สำหรับผู้มาเยือนอีกครั้งเพื่อยืนยัน จากนั้นกดปุ่ม [Registration] เพื่อเสร็จสิ้นขั้นตอน

:::note[หมายเหตุ]
หากคุณลงทะเบียนรหัส PIN สำหรับผู้มาเยือนใหม่ รหัสเก่าจะถูกลบออกโดยอัตโนมัติ
:::

:::note[หมายเหตุ]
สามารถบันทึกรหัส PIN สำหรับผู้มาเยือนได้เพียง 1 ชุดเท่านั้น และจะถูกลบออกโดยอัตโนมัติหลังจากใช้งาน 1 ครั้ง
:::

## Dual-mode Security Setting

### ขั้นตอน

1. หลังจากป้อนรหัส PIN ผู้ใช้แล้ว ให้กดปุ่ม [#] บนหน้าจอ LED สองครั้ง เพื่อเปิดโหมดตั้งค่าคุณสมบัติพื้นฐาน
2. กดปุ่ม LED [2] หรือกดปุ่ม LED [5] แล้วตามด้วยปุ่ม [Registration] เพื่อเสร็จสิ้นการตั้งค่า

:::note[หมายเหตุ]
LED [2]: เปิดใช้งานการยืนยันตัวตนแบบสองชั้น
:::

:::note[หมายเหตุ]
LED [5]: ปิดการใช้งานการยืนยันตัวตนแบบสองชั้น
:::

:::note[หมายเหตุ]
รีโมทคอนโทรล, รหัส PIN สำหรับผู้มาเยือน และระบบ Bluetooth สามารถสั่งปลดล็อกเปิดสลักกลอนได้โดยไม่ต้องยืนยันตัวตนขั้นที่สอง
:::

:::note[หมายเหตุ]
หากข้อมูลการยืนยันสิทธิ์ขั้นสอง (บัตรหรือลายนิ้วมือ) ยังไม่ลงทะเบียน ระบบจะปลดล็อกเปิดประตูด้วยรหัสผ่าน PIN เท่านั้น
:::

:::note[หมายเหตุ]
หากยังไม่ได้ลงทะเบียนระบบยืนยันตัวตนขั้นที่สองของผู้ใช้ จะไม่สามารถเปิดใช้งานการยืนยันตัวตนแบบสองชั้นได้ และปุ่ม LED [2] จะกะพริบพร้อมเสียงเตือนข้อผิดพลาด
:::

:::note[หมายเหตุ]
เมื่อเปิดใช้โหมดยืนยันตัวตนสองชั้นร่วมกับบัตรหรือลายนิ้วมือ หากข้อมูลการยืนยันสิทธิ์ขั้นสองเหล่านี้ถูกลบออกไป ระบบจะยกเลิกการใช้โหมดยืนยันตัวตนสองชั้นโดยอัตโนมัติ
:::

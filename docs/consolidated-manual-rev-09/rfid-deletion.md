# การลบบัตร Smart Card

## 🗑 ลบบัตรทั้งหมด (LED [2] — All at once)

**ขั้นตอน:**
1. กดปุ่ม **OPEN/CLOSE** ค้างไว้ **3 วินาที**
2. ใส่ PIN หลัก + `[#]`
3. LED [2] [3] [5] [0] จะติด → กดหมายเลข **[2]** (Card)
4. กดค้าง **`[#]` 5 วินาที** → ได้ยินเพลง → บัตรทั้งหมดถูกลบ

## 🗑 ลบบัตรทีละใบ (LED [2] — One-by-one)

**ขั้นตอน:**
1. กด **OPEN/CLOSE** 3 วินาที → ใส่ PIN + `[#]`
2. LED [2] [3] [5] [0] ติด → กด **[2]**
3. ใส่ **หมายเลขช่อง 3 หลัก** (001-200) ของบัตรที่ต้องการลบ
4. กด `[#]` → ได้ยินเพลง → บัตรนั้นถูกลบ
5. ทำซ้ำสำหรับบัตรอื่น หรือกด **R** เพื่อจบ

## ⚠️ ข้อควรระวัง

:::warning
- **การลบบัตรไม่สามารถยกเลิกได้** — ต้องลงทะเบียนใหม่หากต้องการใช้บัตรอีกครั้ง
- **ลบบัตรแล้ว PIN หลักยังอยู่** — ไม่ลบการตั้งค่าอื่นๆ
- ถ้าไม่มีบัตรที่ต้องการลบ กด **R** เพื่อออก
- ถ้าเกิดข้อผิดพลาด LED [2] จะกระพริบ
- ถ้า Smart Card Auto Recognition เปิดอยู่ บัตรจะถูกอ่านทันที — ต้องปิด Auto ก่อนลบ
:::

# การลงทะเบียนบัตร Smart Card

## 1. ลงทะเบียนบัตร (LED [2])

**รองรับสูงสุด 200 ใบ** ต่อล็อก

**ขั้นตอน:**
1. กดปุ่ม **R** แล้วใส่ PIN หลัก + `[#]`
2. LED [1] [2] [3] [4] [5] จะติด → กดหมายเลข **[2]**
3. แตะบัตร Smart Card ที่เครื่องอ่าน → ได้ยินเสียงบี๊บ
4. กดปุ่ม **R** → ได้ยินเพลงยืนยัน

:::tip
- บัตร Smart Card ต้องเป็นแบบ **13.56 MHz** (FeliCa/Mifare)
- สามารถลงทะเบียนได้สูงสุด **200 ใบ**
- บัตรที่ลงทะเบียนแล้วจะแสดงในระบบจัดการ
:::

## 2. Smart Card Auto Recognition / Manual Recognition (LED [1] / [4])

ตั้งค่าโหมดการอ่านบัตร Smart Card

**ขั้นตอน:**
1. สัมผัสคีย์แพดเพื่อเปิดไฟ LED
2. ใส่ PIN หลัก แล้วกด `[#]`
3. กดหมายเลข **[1]** สำหรับ **Auto** หรือ **[4]** สำหรับ **Manual**
4. กดปุ่ม **R** → ได้ยินเพลงยืนยัน

| โหมด | พฤติกรรม |
|---|---|
| **Auto (LED [1])** | วางบัตรบนเครื่องอ่าน → ประตูเปิดทันที |
| **Manual (LED [4])** | ต้องแตะคีย์แพดก่อน แล้ววางบัตร |

:::caution
- ในโหมด **Manual** ถ้าไม่แตะคีย์แพดก่อน จะอ่านบัตรไม่ได้
- เมื่อตั้งค่า Auto แล้ว LED [2] จะกระพริบเมื่อ FeliCa Auto-Detection ถูกปิด
- LED [2] กระพริบ + เสียงเตือน = ข้อผิดพลาด
:::

## 3. Door Open Alarm ON/OFF (LED [7])

ตั้งค่าเสียงเตือนเมื่อประตูเปิดค้าง

**ขั้นตอน:**
1. สัมผัสคีย์แพด → ใส่ PIN + `[#]`
2. กด **[7]** เพื่อ toggle ON/OFF
3. กด **R** เพื่อยืนยัน

**พฤติกรรม:**
- **เปิดใช้งาน:** เมื่อ Door Open Alarm Mode เปิด และ Deadbolt Auto Lock ทำงาน ถ้า Latch/Magnet sensor ถูกตัดการเชื่อมต่อ → **เสียงบี๊บ 3 ครั้ง ทุก 7 วินาที**

## ⚠️ ข้อควรระวัง

- ถ้าลงทะเบียนบัตรเกิน 200 ใบ ล็อกจะปฏิเสธ — ต้องลบบัตรเก่าก่อน
- การลงทะเบียน Card ไม่ลบ PIN หลัก
- ถ้าต้องการลบบัตรทั้งหมด ใช้ **LED [2] ในโหมด Deletion** (กด 5 วินาที)
- ถ้าเกิดข้อผิดพลาด LED [2] จะกระพริบ
- บัตรที่ใช้งานร่วมกับ **FeliCa Auto-Detection** จะอ่านอัตโนมัติเมื่ออยู่ในระยะ
- **Smart Card + PIN** = Dual Authentication

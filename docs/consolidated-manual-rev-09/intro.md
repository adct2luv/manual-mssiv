## ✨ คุณสมบัติเด่น

- **Dual Authentication** (การยืนยันตัวตนสองชั้น) — เมื่อเปิดใช้ ต้องยืนยันด้วย PIN + (Card หรือ Fingerprint)
- **Internal / External Forced Lock** — ล็อกบังคับจากด้านใน/นอก
- **Smart Card Auto Recognition** — บัตร RFID จดจำอัตโนมัติ
- **Random Number Feature** — สุ่มตัวเลขก่อน/หลัง PIN เพื่อป้องกันการแอบมอง
- **1-Minute Lock Warning** — ล็อก 1 นาทีหลังใส่รหัสผิด 5 ครั้ง
- **Low Battery Warning** — เตือนเมื่อแบตเตอรี่ต่ำ
- **Automatic Re-locking** — ล็อกอัตโนมัติหลังประตูปิด
- **Deadbolt Error Warning** — เตือนเมื่อกลอนตายทำงานผิดปกติ
- **Intrusion Alarm** — สัญญาณเตือนการบุกรุก

# การลบลายนิ้วมือ

## 🗑 ลบทั้งหมด (LED [3] — All at once)

:::warning
การลบแบบนี้จะลบลายนิ้วมือ **ทั้งหมด 100 ช่อง** ในครั้งเดียว
:::

**ขั้นตอน:**
1. กด **OPEN/CLOSE** 3 วินาที → ใส่ PIN + `[#]`
2. LED [2] [3] [5] [0] ติด → กด **[3]** (Fingerprint)
3. กดค้าง **`[#]` 5 วินาที** → ได้ยินเพลง → ลายนิ้วมือทั้งหมดถูกลบ

## 🗑 ลบทีละช่อง (LED [3] — One-by-one)

**ขั้นตอน:**
1. กด **OPEN/CLOSE** 3 วินาที → ใส่ PIN + `[#]`
2. LED [2] [3] [5] [0] ติด → กด **[3]**
3. กด `[#]` → ตัวเลขบนคีย์แพดติดทั้งหมด (standby mode)
4. ใส่ **หมายเลขช่อง 3 หลัก** (001-100) ของลายนิ้วมือที่ต้องการลบ
5. กด `[#]` → ได้ยินเพลง → ลายนิ้วมือนั้นถูกลบ
6. ทำซ้ำสำหรับลายอื่น หรือกด **R** เพื่อจบ

## ⚠️ ข้อควรระวัง

:::warning
- **การลบลายนิ้วมือไม่สามารถยกเลิกได้** — ต้องลงทะเบียนใหม่หากต้องการใช้
- **ลบแล้ว PIN หลักยังอยู่** — ไม่ลบการตั้งค่าอื่น
- ถ้าไม่มีลายนิ้วมือที่ต้องการลบ กด **R** เพื่อออก
- ถ้าเกิดข้อผิดพลาด LED [0] จะติด
- **5 ครั้งที่สแกนผิดติดต่อกัน** = Anti-Prank + ล็อก 1 นาที
- ต้องลงทะเบียนลายนิ้วมือ **ผ่านก่อน** จึงจะลบได้
- **ลายนิ้วมือจะถูกลบทั้งหมด** ถ้าใช้ LED [3] แบบ All at once
:::

# คุณสมบัติพิเศษ (Convenient Features)

## 🔢 Random Number Feature (LED [9])

เพิ่มตัวเลขสุ่มก่อน/หลัง PIN เพื่อป้องกันการแอบมอง

**กติกา:**
- ใส่ตัวเลขได้สูงสุด **20 หลัก** (รวม PIN)
- **ต้องใส่ PIN หลังจาก** Random Number
- ตัวอย่าง: `2580` + `1234` + `*` (ใส่ 2580 ก่อน 1234)

**ประโยชน์:**
- ป้องกันคนแอบมอง PIN
- ลบรอยนิ้วมือบนคีย์แพด
- ใช้ได้ทั้ง PIN และ Card

## 🔒 Automatic Re-locking Function

**พฤติกรรม:**
- ถ้าประตูไม่ได้เปิดภายใน **7 วินาที** หลังจากปลดล็อก → ล็อกอัตโนมัติ
- ใช้ได้กับทั้ง PIN, Card, Fingerprint
- ไม่ต้องตั้งค่า — เปิดใช้งานเป็นค่าเริ่มต้น

**ตัวอย่าง:**
- เปิดประตูด้วย PIN → ไม่เปิดจริงภายใน 7 วินาที → ล็อกอัตโนมัติ
- กด OPEN/CLOSE → ประตูปิดและล็อกทันที

## ⏰ 1 Minute Lock Warning Function

:::warning
**5 ครั้งติดต่อกัน** ที่ใส่รหัส/แตะบัตร/สแกนนิ้วผิด → **ล็อก 1 นาที**
:::

**พฤติกรรม:**
- **หลัง PIN/Card/Fingerprint ผิด 5 ครั้ง:**
  - เสียงเตือนดัง **7 วินาที**
  - ล็อกหยุดทำงาน **1 นาที**
  - ระหว่างล็อก → บัตร/PIN/Remote Control ใช้ไม่ได้
- **ยกเลิกได้:** กดปุ่ม **OPEN/CLOSE**
- **หลัง 1 นาที:** ล็อกกลับสู่สถานะปกติ

## 🔋 Low Battery Warning Function

**เมื่อแบตเตอรี่ต่ำ:**
- LED **[Battery Replacement]** ติด
- เสียง "beep-ree-reep" ดัง
- การยืนยันตัวตนอาจทำงานผิดปกติ

**เกณฑ์:**
- แรงดันต่ำ: **4.5V** (Face Recognition Door Lock: 5.0V)
- เมื่อแรงดันกลับสูง **5.5V** (Face: 5.6V) → สัญญาณเตือนจะหายไป

:::caution
- **เปลี่ยนแบตเตอรี่ทั้งหมดพร้อมกัน** อย่าผสมเก่า+ใหม่
- ใช้แบตเตอรี่ Alkaline AA **4 ก้อน**
- ไฟฉุกเฉิน: 9V Alkaline (ไม่รวม) หรือ USB C-Type (รุ่น F7000Kr/F9000Kr/P8800K)
:::

## 🔇 Etiquette Mode (LED [9])

โหมดเงียบสำหรับการใช้งาน 1 ครั้ง

**ขั้นตอน:**
1. แตะคีย์แพด
2. กด `[#]`

**ผลกระทบ:**
- **การยืนยัน PIN, Card, Fingerprint, Face, Guest PIN** ครั้งถัดไปจะเงียบ
- ใช้ได้ครั้งเดียว แล้วเสียงกลับมาปกติ
- **ถ้ายืนยันตัวตนล้มเหลว 5 ครั้งติดต่อกัน** → เสียงเตือนดัง

## 🌙 Sound Mute (LED [9]) — สำหรับ 1 ครั้ง

เปิดใช้งานโดยกด `[*]` ในสถานะปกติ → เสียงจะเงียบสำหรับการใช้งาน 1 ครั้ง

## ⚠️ Etiquette vs Sound Mute

| คุณสมบัติ | ระยะเวลา | วิธีเปิด |
|---|---|---|
| **Etiquette Mode** | 1 ครั้ง (หลังเปิด) | กด `[#]` ก่อน |
| **Sound Mute** | 1 ครั้ง (ถัดไป) | กด `[*]` ก่อน |
| **Voice Mode** | ถาวร | ตั้งค่าผ่าน LED |
| **Buzzer Mode** (default) | ถาวร | ตั้งค่าผ่าน LED |

## 🪟 Automatic Re-locking จากด้านใน

- **Sub Type (ประตูบานเลื่อน):** กดปุ่ม OPEN/CLOSE + หมุนปุ่ม
- **Main Type:** กดปุ่ม OPEN/CLOSE
- **เมื่อ Deadbolt Auto Lock เปิด:** แตะคีย์แพด 2 วินาที → ล็อกอัตโนมัติ
- **ถ้า Deadbolt Auto Lock ปิด:** แตะคีย์แพด 2 วินาที → ล็อกด้วยตนเอง

## 🔓 การปลดล็อกจากด้านใน

| สถานะ | วิธีปลดล็อก |
|---|---|
| **ไม่มี Deadbolt** | ดึงที่จับด้านใน |
| **มี Deadbolt Auto Lock** | กด OPEN/CLOSE 2 วินาที (ปลดล็อกอัตโนมัติ) |
| **มี Deadbolt Manual Lock** | แตะคีย์แพด 2 วินาที หรือ กด OPEN/CLOSE เพื่อปลดล็อก |

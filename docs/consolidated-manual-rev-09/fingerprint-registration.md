# การลงทะเบียนลายนิ้วมือ

:::info ใช้ได้กับรุ่นที่มี Fingerprint Reader
ES-F300Dr, ES-F301D, ES-F500D/H, ES-F501D/H, ES-S510D/H, ES-FF730Gr, ES-FF731G, ES-S740D, ES-F7000Kr, ES-F9000Kr, ES-P8800K
:::

**รองรับสูงสุด 100 ลายนิ้วมือ** ต่อล็อก

## 🖐 ลงทะเบียนลายนิ้วมือ (LED [3])

**ขั้นตอน:**
1. กดปุ่ม **R** แล้วใส่ PIN หลัก + `[#]`
2. LED [1] [2] [3] [4] [5] จะติด → กดหมายเลข **[3]** (Fingerprint)
3. **วางนิ้วบนเซ็นเซอร์ 4 ครั้ง** (ครั้งละ ~1 วินาที) — ได้ยินเสียงบี๊บแต่ละครั้ง
4. เมื่อสำเร็จครั้งที่ 4 → ได้ยินเพลง → ลายนิ้วมือลงทะเบียนสำเร็จ
5. กดปุ่ม **R** เพื่อเสร็จสิ้น

## 📊 LED แสดงสถานะการสแกน

| LED | ความหมาย |
|---|---|
| LED [1] [2] [3] กระพริบ [2] | สแกนครั้งที่ 1 สำเร็จ |
| LED [1] [2] [3] กระพริบ [3] | สแกนครั้งที่ 2 สำเร็จ |
| LED [1] [2] [3] กระพริบ (ไม่ตัว) | สแกนครั้งที่ 3 สำเร็จ |
| LED [0] ติด | ข้อผิดพลาด หรือนิ้วไม่ตรง |

## 💡 Tips สำหรับการสแกนลายนิ้วมือ

- **วางนิ้วให้เต็มเซ็นเซอร์** — ไม่ใช่ปลายนิ้ว
- **นิ้วต้องสะอาด** ไม่เปียก ไม่มีครีม ไม่มีแผล
- **กดนิ้วเบาๆ** — ไม่ต้องกดแรง
- ลายนิ้วมือเด็กหรือผู้สูงอายุอาจสแกนยาก — แนะนำให้ใช้วิธียืนยันตัวตนอื่นร่วมด้วย

## ⚠️ ข้อควรระวัง

:::warning
- **ลายนิ้วมือเดียวต่อผู้ใช้** — ไม่ควรลงทะเบียนนิ้วเดียวกัน 2 ครั้ง
- ถ้า **สแกนไม่สำเร็จ 4 ครั้ง** → LED [0] ติด → ต้องเริ่มใหม่
- **ลายนิ้วมือจะถูกลบทั้งหมด** ถ้าลบผ่าน LED [3] ในโหมด Deletion
- **การลงทะเบียนลายนิ้วมือไม่ลบ PIN หลัก**
- **5 ครั้งที่สแกนผิดติดต่อกัน** = Anti-Prank + ล็อก 1 นาที
- ต้องลงทะเบียน **Fingerprint ผ่าน** ก่อนใช้ Dual Authentication
- ลายนิ้วมือที่ชำรุด/แผลเป็น/เปียก อาจสแกนไม่ติด → ใช้ PIN แทน
:::

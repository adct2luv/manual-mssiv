# การลงทะเบียนรหัส PIN

## 1. ลงทะเบียน PIN หลัก (LED [1])

PIN หลักใช้เปิดประตู + เป็นรหัสผ่านสำหรับเข้าโหมดลงทะเบียน/ลบ

**ขั้นตอน:**
1. เปิดฝาครอบแบตเตอรี่ด้านใน
2. กดปุ่ม **Registration (R)**
3. ใส่ **PIN เริ่มต้น `1234`** แล้วกด `[#]`
4. LED [1] จะติดบนคีย์แพด
5. ใส่ **PIN ใหม่** (4-12 หลัก)
6. กดปุ่ม **R** → ได้ยินเพลงยืนยัน

:::caution ก่อนเริ่ม
- ลงทะเบียนขณะประตูเปิดเสมอ
- รหัส PIN เริ่มต้นคือ `1, 2, 3, 4` (ต้องเปลี่ยนครั้งแรก)
- **มี PIN ได้แค่ 1 ชุด** — การลงทะเบียนใหม่จะลบของเก่า
:::

## 2. ลงทะเบียน Guest PIN (LED [4])

Guest PIN ใช้ได้ **1 ครั้ง** เหมาะสำหรับแขกผู้มาเยือน

**ขั้นตอน:**
1. กดปุ่ม **R** แล้วใส่ PIN หลัก + `[#]`
2. กดหมายเลข **[4]** — LED [4] ติด
3. ใส่ **Guest PIN ใหม่** (4-12 หลัก)
4. กดปุ่ม **R** → ได้ยินเพลงยืนยัน

:::caution
- **Guest PIN ใช้ได้แค่ 1 ครั้ง** แล้วจะถูกลบอัตโนมัติ
- **Guest PIN ไม่ต้องยืนยันตัวตน 2 ชั้น** — ใช้ได้ทันที
- **การลงทะเบียน Guest PIN ใหม่จะลบ Guest PIN เก่า**
:::

## 3. ลงทะเบียน Super Admin PIN (R button)

Super Admin PIN ใช้รีเซ็ตระบบและจัดการขั้นสูง

**ขั้นตอน:**
1. กดปุ่ม **OPEN/CLOSE** ค้างไว้ **10 วินาที**
2. ใส่ **PIN หลักปัจจุบัน** แล้วกด `[#]`
3. กดปุ่ม **R** (Registration)
4. ใส่ **Super Admin PIN ใหม่**
5. กดปุ่ม **R** อีกครั้งเพื่อยืนยัน
6. ได้ยินเพลงยืนยัน

## 📋 ข้อควรระวัง

:::warning ข้อผิดพลาดที่อาจเกิด
- **ไม่สามารถลงทะเบียน PIN ที่รวมตัวเลขสุ่ม** (Random Number) ในโหมด Registration
- **ไม่สามารถลงทะเบียน PIN ที่มีอยู่แล้วซ้ำ**
- ต้องลงทะเบียน **Super Admin PIN ใหม่** ก่อนลงทะเบียนอื่นๆ ได้
- ถ้ากด **R button ระหว่างขั้นตอน** การลงทะเบียนจะถูกยกเลิก
:::

## 💡 Tips

- ใช้ **PIN 4 หลัก** ง่าย แต่ถ้าต้องการความปลอดภัยมากขึ้นใช้ **8-12 หลัก**
- ห้ามใช้ PIN ที่เดาง่าย เช่น `1234`, `0000`, ปีเกิด, เบอร์โทร
- ควรเปลี่ยน PIN ทุก 3-6 เดือน
- จด PIN เก็บไว้ในที่ปลอดภัย (ไม่ใช่ข้างล็อก)

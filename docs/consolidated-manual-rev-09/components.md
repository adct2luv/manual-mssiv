# ส่วนประกอบ

## 🏠 ส่วนประกอบตัวล็อกด้านนอก (Outer body)

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **Number Pad** | ใส่รหัส PIN |
| 2 | **Low Battery Lamp** | ไฟแสดงเมื่อแบตเตอรี่ต่ำ |
| 3 | **Reset Button** | รีเซ็ตล็อกเป็นค่าโรงงาน |
| 4 | **Smart Card Reader** | แตะบัตรที่ลงทะเบียนไว้ |
| 5 | **Fingerprint Reader** | แตะนิ้วที่ลงทะเบียนไว้ (รุ่นที่รองรับ) |
| 6 | **Face Recognition Camera** | มองตรงเข้ากล้อง (รุ่นที่รองรับ) |
| 7 | **IR Sensor** | ตรวจจับการเข้าใกล้ |
| 8 | **Emergency Battery Terminal** | แตะแบต 9V เมื่อแบตหมด |
| 9 | **Mechanical Key Hole** | ใช้กุญแจฉุกเฉิน (รุ่นที่มี) |
| 10 | **Outer Body Handle** | ดึงลงเพื่อเปิดประตูจากด้านนอก |

## 🔧 ส่วนประกอบตัวล็อกด้านใน (Inner body)

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **Open/Close Button** | เปิด/ปิดประตูจากด้านใน |
| 2 | **Registration Button (R)** | เข้าโหมดลงทะเบียน/ลบ |
| 3 | **Manual Lock Button** | ล็อกด้วยตัวเอง |
| 4 | **Smart Card Lamp** | ไฟแสดงสถานะบัตร |
| 5 | **Smart Card Reader (Inner)** | เครื่องอ่านบัตรด้านใน |
| 6 | **Emergency Battery Terminal** | ช่องเสียบแบต 9V |
| 7 | **Battery Cover** | ฝาครอบช่องใส่แบตเตอรี่ |
| 8 | **Inner Body Handle** | ดึงลงเพื่อเปิดประตูจากด้านใน |
| 9 | **Battery (AA × 4)** | แหล่งพลังงานหลัก |

## 🔌 อุปกรณ์เสริม (Optional Accessories)

| อุปกรณ์ | รายละเอียด |
|---|---|
| **Home Network Module** | เชื่อมต่อ Wi-Fi เพื่อควบคุมจากระยะไกล |
| **Remote Control** | รีโมตเปิดประตู (RX / TRX) |
| **RX, BLE** | รุ่นใหม่ใช้ Bluetooth (ES-P8800K) |
| **Remote Control (BLE)** | รีโมต Bluetooth (ES-P8800K) |

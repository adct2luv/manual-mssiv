## 📌 รายละเอียดเพิ่มเติม

### ขนาดประตูที่รองรับ
- **30~50mm:** ES-S100Dr, ES-F300Dr, ES-F301D, ES-F500D/H, ES-F501D/H, ES-S510D/H, ES-S740D, ES-FF730Gr, ES-FF731G, ES-F7000Kr
- **40~50mm:** ES-F9000Kr, ES-P8800K

### แหล่งพลังงาน
- ใช้ถ่าน AA Alkaline **4 ก้อน** (DC 6V) ทุกรุ่น
- ไฟฉุกเฉิน: แบตเตอรี่ 9V Alkaline (ไม่รวมในกล่อง) — ยกเว้น **ES-F7000Kr / ES-F9000Kr / ES-P8800K** ที่มี **USB C-Type** สำหรับไฟฉุกเฉิน

### อุณหภูมิตรวจจับไฟ
- **60°C ± 10°C** — ทุกรุ่น

### วัสดุ
- **Zn, Al, ABS** — ทุกรุ่น

## 🔧 คุณสมบัติทั่วไปทุกรุ่น

:::info ทุกรุ่นในคู่มือนี้มี
- **การลงทะเบียน PIN** (LED [1])
- **การลงทะเบียนบัตร Smart Card** (LED [2]) — สูงสุด 200 ใบ
- **Smart Card Auto Recognition / Manual Recognition** (LED [1] / [4])
- **การตรวจจับอัตโนมัติ** (Smart Card + ใบหน้า)
- **สัญญาณเตือนความปลอดภัย** (High Temp, Intrusion, Deadbolt, Anti-Prank, 1-minute lock)
- **การล็อกอัตโนมัติ** (Auto Re-locking หลัง 7 วินาที)
- **รหัสสุ่ม** (Random Number ก่อน/หลัง PIN)
- **Dual Authentication** (PIN + Card/Fingerprint/Face)
- **ฟังก์ชัน Forced Lock** (ภายใน/ภายนอก)
- **กุญแจกลไกฉุกเฉิน** (เฉพาะรุ่นที่มี)
- **ไฟแบตเตอรี่ต่ำ** (Battery Replacement LED)
- **การเชื่อมต่อ IR Sensor** (Face recognition)
- **การเชื่อมต่อ Home Network / Remote Control** (อุปกรณ์เสริม)
:::

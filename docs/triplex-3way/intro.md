# TRIPLEX 3way — กุญแจดิจิทัล TRIPLEX 3way

## คุณสมบัติเด่น

:::warning ต้องติดตั้งโดยช่างผู้เชี่ยวชาญ
ผลิตภัณฑ์นี้ต้องติดตั้งโดย **ช่างผู้เชี่ยวชาญเท่านั้น** เพื่อให้ได้รับการรับประกันเต็มรูปแบบ ห้ามติดตั้งด้วยตัวเอง
:::

# คุณสมบัติพิเศษ

## Multi-Touch Feature

### ขั้นตอนการตั้งค่า
1. Touch the [Number pad] on the Outer body. ป้อนรหัส PIN, กดปุ่ม [✱].
2. Press the [5] button on the number pad, a melody will be heard and Multi-touch feature setting เสร็จสมบูรณ์.

:::tip[วิธียกเลิก]
To cancel the setting, just repeat steps 1, 2.
:::

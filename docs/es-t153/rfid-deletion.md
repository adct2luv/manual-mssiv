# การลบบัตร RFID

## ลบทั้งหมด
1. เปิดฝาครอบแบตเตอรี่ กด **Registration**
2. ใส่รหัส PIN แล้วกด `*`
3. กดปุ่มลบ
4. กด `*` ค้าง 5 วินาที — เมื่อได้ยินเพลง บัตรถูกลบ

## ลบทีละใบ
1. ทำตามขั้นตอน 1-3 ข้างต้น
2. ใส่หมายเลขช่อง 3 หลัก แล้วกด `#`

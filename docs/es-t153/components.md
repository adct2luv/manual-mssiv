# ส่วนประกอบ

## ตัวล็อกด้านนอก
<a id="ขั้วแบตเตอรี่ฉุกเฉิน"></a>

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **ปุ่มรีสตาร์ทเครื่อง (Reset)** | ใช้สำหรับรีสตาร์ทระบบกรณีตัวเครื่องขัดข้องหรือไม่ตอบสนอง |
| 2 | **Status LED lamp** |  |
| 3 | **Number pad** |  |
| 4 | **Battery lamp** |  |
| 5 | **เครื่องอ่านการ์ด RFID** | แตะบัตรหรือแท็กคีย์การ์ดที่ลงทะเบียนแล้วเพื่อเปิดประตู |
| 6 | **ขั้วต่อแบตเตอรี่ฉุกเฉิน (9V)** | แตะแบตเตอรี่ 9V เพื่อจ่ายไฟสำรองหากแบตเตอรี่หลักหมด |

## ตัวล็อกด้านใน

| # | ชิ้นส่วน | หน้าที่ |
|---|---|---|
| 1 | **สลักล็อก (Deadbolt)** | สลักล็อกหลักของประตูเมื่อประตูล็อก |
| 2 | **Door closing sensor** |  |
| 3 | **Manual lock knob** |  |
| 4 | **ปุ่มเปิด/ปิดประตู (Open/Close)** | กดปุ่มเพื่อล็อกหรือปลดล็อกประตูแบบสัมผัสเดียวจากด้านใน |
| 5 | **ปุ่มลงทะเบียน (Registration)** | ใช้กดเมื่อต้องการลงทะเบียนหรือแก้ไขรหัส PIN, บัตร หรือลายนิ้วมือ |
| 6 | **Alkaline AA battery (×4)** |  |
| 7 | **ฝาครอบแบตเตอรี่** | เปิดออกเพื่อเข้าถึงช่องใส่แบตเตอรี่ |

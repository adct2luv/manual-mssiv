# Components

## Outer body

| # | Part | Function |
|---|---|---|
| 1 | **Fingerprint Reader** | Scan registered fingerprint here |
| 2 | **Fingerprint Cover Open Button** | Press to open fingerprint cover |
| 3 | **RFID Card Reader** | Tap registered card here |
| 4 | **Function sensor bar** | Touch to activate keyboard/card sensor |
| 5 | **Number pad** |  |
| 6 | **Battery lamp** |  |
| 7 | **Mechanical Key Hole** | Insert emergency mechanical key here |
| 8 | **Emergency Battery Terminal** | Touch 9V battery here if lock is dead |
| 9 | **Reset Button** | Reset lock to factory defaults (rarely used) |
| 10 | **Status LED Lamp** | Shows lock status (idle / locked / error) |
| 11 | **Outer Body Handle** | Pull down to retract bolt after unlock |

## Inner body

| # | Part | Function |
|---|---|---|
| 1 | **Battery Cover** | Slide off to access battery compartment |
| 2 | **Receiver module attachment (Optional)** | Slot to insert remote control/home network module |
| 3 | **Registration Button** | Used when registering PIN, Card or Fingerprint |
| 4 | **Mortise** |  |
| 5 | **Deadbolt** | Locking bolt that secures the door |
| 6 | **Latch Bolt** | Spring-loaded latch bolt |
| 7 | **Internal handle lock button** | Locks the internal handle for safety |
| 8 | **Alkaline AA Battery** | Main power source |
| 9 | **Close button (Double lock 3 seconds)** | Closes the door or enables double lock |
| 10 | **Speaker** | Outputs voice guidance and alarm sounds |
| 11 | **Inner Body Handle** | Pull down to retract bolt from inside |

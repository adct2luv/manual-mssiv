# IR Sensor Usage Guide for Face ID

Technical reference and installation guide for the infrared movement sensor (IR Sensor) used to trigger Face ID in models **ES-S740D** and **ES-S510D/H**.

## 📡 Auto Detection (IR Sensor) Overview
The IR sensor detects people approaching the door to wake up the Face ID sensor automatically, providing a completely touchless entry experience.

### Detection Distance by Environment:
1. **WC Mirror (High Reflection):**
   - **Detection Distance:** ~1.75 meters
   - **Testing Height:** 1.2 meters
   - **Result:** Detects up to 1.75m due to mirror reflectivity.
2. **Person (Human Body):**
   - **Detection Distance:** ~0.9 meters
   - **Result:** Wakes up accurately at 0.9m when a person walks towards the lock.
3. **White Board / Other Reflective Surfaces:**
   - **Detection Distance:** ~1.25 meters
   - **Result:** Reflection from whiteboard surfaces triggers detection at 1.25m.

---

## ⚠️ Pre-Installation Environment Check

:::warning[Important Environment Warning]
- The test results and distances mentioned above are for reference only. The actual detection ranges will depend heavily on your specific door installation surroundings.
- Check the installation site beforehand.
- If the door lock is facing a wall closely or placed in a wall-to-wall corridor, reflection may cause unnecessary wake-ups. We request that you disable the IR sensor using the **IR Sensor ON/OFF settings** (refer to the device manual).
:::

# EPIC Things App — Quick Guide

The **EPIC Things** app controls and manages EPIC digital locks via smartphone. Supports **Bluetooth (BLE)** and **Wi-Fi Bridge** for remote control.

## 📱 Download the app

- **iOS:** App Store → search "EPIC Things"
- **Android:** Google Play Store → search "EPIC Things"

QR code for download is on page 3 of the PDF.

## 🎯 What's needed

:::tip Super Admin PIN is required
A **Super Admin PIN** is needed before using the EPIC Things app.

**What is Super Admin?**
EPIC Door Lock has 3 user levels:
- **Super Admin** (highest level) — links app to lock, manages all lock functions
- **General Admin**
- **User** (regular user)

**Super Admin privileges:**
- Full access to all lock functions
- Authority to link the app with the door lock (prevents non-owners from linking)
- Add/remove members, manage settings
:::

## ✨ Key Features

- **Bluetooth (BLE) control** — works within range
- **Wi-Fi Bridge control** — works remotely
- **Member management** — add/remove users, set permissions
- **Key management** — PIN, Card, Fingerprint, Remote, Face ID
- **View history** — all entry/exit events
- **Guest Key** — temporary access for visitors
- **Bluetooth + Wi-Fi** — switch connection modes

:::warning Cautions
- **Super Admin PIN must be set first** before using the app
- **Store Super Admin PIN safely** — if forgotten, contact a technician
- **Bluetooth works only nearby** — Wi-Fi Bridge required for Guest Key (remote)
- **Cannot register PIN directly through the app** — must be done at the lock
:::

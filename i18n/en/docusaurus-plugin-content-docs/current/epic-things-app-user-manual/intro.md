# EPIC Things APP — User Manual

The **EPIC Things** mobile app lets you **control and manage EPIC digital door locks** from your smartphone. Available for both iOS and Android. Uses **Bluetooth (BLE)** for nearby control and **Wi-Fi Bridge** for remote access.

## 📱 Key Features

- **Super Admin PIN** — Master password for the administrator
- **Guest Login** — Temporary user login
- **Member Management** — Manage members and access permissions
- **Door Lock Registration** — Register digital locks with the app
- **WiFi Bridge Management** — Manage Wi-Fi connectivity
- **Key Management** — Manage PIN, Card, Fingerprint, Remote Control, Face ID
- **History Management** — View entry/exit history

## ⚠️ Warnings

- **Do not use rechargeable batteries** in locks managed via the app
- **Do not expose the lock to direct water**
- **Keep your Wi-Fi password secret** — anyone on the same Wi-Fi network can control the lock

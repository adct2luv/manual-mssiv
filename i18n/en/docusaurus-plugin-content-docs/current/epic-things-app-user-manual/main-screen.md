# Main Screen

The EPIC Things app's main screen has 6 main sections:

## 1. 🔓 Open/Close

**Use this button to open or close the door.** When the door is open, the lock icon in the center changes to an **open state** (🔓).

## 2. 📊 Door Lock Status

**Check the current lock status** — shows the lock's status. In certain situations, the icon will appear in red.

**Statuses shown:**
- 🔓 **Door is open**
- ⚠️ **Low Battery Alarm**
- ⚠️ **Trick Alarm**
- ⚠️ **Spring Alarm**
- ⚠️ **High-temperature Alarm**

## 3. 🔐 Door Lock List

Shows **the locks currently in use** — select to view a list of available locks.

- The currently used door lock has a red border
- The lock registered to your account is labeled "me"

## 4. 📡 Connection Mode

Shows **the current connection mode** — select to view available modes.

**Supported modes:**
- 🔵 **Bluetooth (BLE)** — default, works within range
- 🔴 **Wi-Fi** — requires Wi-Fi Bridge, works remotely

:::info
- Bluetooth mode is the **default**
- Wi-Fi mode requires a **Wi-Fi Bridge**
- **Wi-Fi supports Guest Key and remote control**
:::

## 5. 👤 Profile

**Profile settings** — see [App Settings](/en/epic-things-app-user-manual/app-settings) for more details.

## 6. 📋 Menu

**Main menu** includes:
- **Member** — manage members ([Member →](/en/epic-things-app-user-manual/member))
- **Door Lock** — connect/manage locks ([Door Lock →](/en/epic-things-app-user-manual/door-lock))
- **Wifi Bridge** — configure Wi-Fi Bridge ([Wifi Bridge →](/en/epic-things-app-user-manual/wifi-bridge))
- **Key** — manage keys ([Key →](/en/epic-things-app-user-manual/key-management))
- **Guest Key** — guest keys ([Guest Key →](/en/epic-things-app-user-manual/guest-key))
- **History** — entry/exit history ([History →](/en/epic-things-app-user-manual/history))
- **App Settings** — app configuration ([App Settings →](/en/epic-things-app-user-manual/app-settings))

## 🎯 Basic usage

### Open door from the app
1. Tap **Open/Close** (lock icon in center)
2. Lock opens (must be within Bluetooth or Wi-Fi range)
3. Icon changes to **open state**

### Check lock status
1. View **status on Door Lock Status**
2. **Red** = problem (low battery, forced entry, high temperature)

### Switch between Bluetooth / Wi-Fi
1. Tap **Connection Mode**
2. Select Bluetooth or Wi-Fi
3. Wi-Fi requires a **registered Wi-Fi Bridge**

:::tip Getting started
1. Set **Super Admin PIN** at the lock ([Setup](/en/epic-things-app-user-manual/setup))
2. **Login** in the app
3. **Add Door Lock** ([Door Lock →](/en/epic-things-app-user-manual/door-lock))
4. Start using!
:::

# Opening & Closing the Door

## Opening Door Outside

### Opening by PIN Code

1. Touch the number pad to turn on the screen.
2. Enter the registered PIN code (default is 1, 2, 3, 4).
3. Press the [✱] button.
4. Push or pull the door to open.

## Closing Door Outside

### Automatic Lock Mode

The door will automatically lock within 2 seconds of closing the door

### Manual Lock Mode

Touch the number pad, to close the door.

## Opening Door Inside

### Opening by Open/Close Button

1. While the door is locked, press the [Open/Close] button on the inner body.
2. The door will unlock, then push or pull to open.

### Opening by Manual Lock/Unlock Knob

1. Turn the manual lock/unlock knob on the inner body to the [Open] direction.
2. The door will unlock, then push or pull to open.

## Closing Door Inside

### Automatic Lock Mode

The door will lock automatically after 2 seconds of closing the door.

### Manual Lock Mode

In manual mode, the door does not lock automatically when closed.

1. Close the door.
2. Press the [Open/Close] button or turn the manual lock knob/button to lock the door.

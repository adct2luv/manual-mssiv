# Special Features

## Smart Card Auto / Manual Recognition

Set whether the lock detects RFID smart cards automatically or manually.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press [1] for Auto Recognition (LED 1), or [4] for Manual Recognition (LED 4).
4. Press the [Registration] button to complete.

:::note
- Auto Recognition: place the card on the reader and the door opens.
- Manual Recognition: touch the number pad first, then place the card on the reader.
:::

## Deadbolt Auto Lock / Manual Lock

An optional feature that lets you set the deadbolt to lock automatically or manually.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press [2] for Auto Lock (LED 2), or [5] for Manual Lock (LED 5).
4. Press the [Registration] button to complete.

:::note
- When Auto Lock is set, the deadbolt locks automatically based on the door sensor.
- When Manual Lock is set, touch the number pad for 2 seconds to lock.
:::

## Sound Setting (Voice, Buzzer, Warning, Door Open Alarm)

Adjust the operation volume.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press [3] to increase volume (LED 3), or [6] to decrease volume (LED 6).
4. Press the [Registration] button to complete.

:::note
- Controls all volume except Alarm Notification and Door Opening Melody.
- Volume can be set to 8 levels (level 1: silent ~ level 8: 80 dB).
:::

## Voice Mode / Buzzer Mode Setting

Switch between voice guidance and buzzer tones.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press the [8] button.
4. Press the [Registration] button to complete.

:::note
Voice Mode and Buzzer Mode are toggled by repeating these steps. For models without Voice Mode, it is always set to Buzzer Mode.
:::

## Door Open Alarm ON/OFF

Sound an alarm when the door is left open.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press the [7] button.
4. Press the [Registration] button to complete.

:::note
When Door Open Alarm is ON and Deadbolt Auto Lock is set, if the latch or magnet sensor is disconnected the buzzer sounds 3 times every 7 seconds.
:::

## Internal Forced Lock ON/OFF

Prevent any entry from outside (PIN, Card, etc.).

### Turn ON (inside)

1. With the door closed and the deadbolt sensor detected, press the [Open/Close] button for 3 seconds.

### Turn OFF (outside)

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press the [9] button.
4. Press the [#] button to complete.

### Turn OFF (inside)

1. Press the [Open/Close] button.
2. Open the door.

:::note
This function stops outside authentication. It can be set when the latch or magnet sensor reacts to the door closing. Internal Forced Lock can be released from outside.
:::

## External Forced Lock Setting

Disable the [Open/Close] button.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button. (First time: Initial PIN [1234] + [#])
3. Press the [0] button.
4. Press the [#] button to complete.

:::note
This function stops the [Open/Close] button. When normal authentication (such as PIN) is performed, the setting is released.
:::

## Dual Authentication ON/OFF

Require two-factor authentication (PIN + Card/Fingerprint/Face) before unlocking.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button twice. (First time: Initial PIN [1234] + [#])
3. Press [2] for ON (LED 2), or [5] for OFF (LED 5).
4. Press the [Registration] button to complete.

:::note
- When ON, you must enter the User PIN and perform a second authentication.
- Admin PIN/Card/Fingerprint/Face ignore Dual Authentication.
- Remote Control, Guest PIN and Bluetooth can unlock without secondary authentication.
- If the user's secondary authentication is not registered, Dual Authentication cannot be turned ON.
:::

## Alarm Notification ON/OFF

Turn the alarm notification on or off.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Enter the User PIN followed by the [#] button twice. (First time: Initial PIN [1234] + [#])
3. Press the [1] button or [4] button.
4. Press the [Registration] button to complete.

:::note
Controls only the Alarm Notification.
:::

## Etiquette Mode

Silent mode for PIN, Card, Fingerprint, Face and Guest entry.

### Steps

1. Touch the number pad to turn on all LEDs.
2. Press the [#] button.

:::note
- Silent mode is applied to PIN, Card, Fingerprint, Face and Guest, for 1 authentication, regardless of the previous Sound Setting.
- A warning volume will occur after 5 consecutive failed authentications.
:::

## Random Number Feature

Enter random digits before the PIN to prevent shoulder-surfing.

:::note
- Up to 20 digits can be entered, including the PIN.
- The PIN must be entered after the random number.
:::

## Automatic Re-locking Function

If the door is not opened within 7 seconds after authentication, the lock re-locks automatically.

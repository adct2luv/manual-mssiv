## 📌 Additional details

### Supported door thickness
- **30~50mm:** ES-S100Dr, ES-F300Dr, ES-F301D, ES-F500D/H, ES-F501D/H, ES-S510D/H, ES-S740D, ES-FF730Gr, ES-FF731G, ES-F7000Kr
- **40~50mm:** ES-F9000Kr, ES-P8800K

### Power source
- 4× AA Alkaline batteries (DC 6V) for all models
- Emergency: 9V Alkaline battery (not included) — except **ES-F7000Kr / ES-F9000Kr / ES-P8800K** which have **USB C-Type** for emergency power

### Fire detection temperature
- **60°C ± 10°C** — all models

### Material
- **Zn, Al, ABS** — all models

## 🔧 Common features for all models

:::info All models in this manual include
- **PIN registration** (LED [1])
- **Smart Card registration** (LED [2]) — up to 200 cards
- **Smart Card Auto/Manual Recognition** (LED [1] / [4])
- **Auto-detection** (Smart Card + Face)
- **Safety alarms** (High Temp, Intrusion, Deadbolt, Anti-Prank, 1-minute lock)
- **Automatic re-locking** (Auto Re-locking after 7 seconds)
- **Random number** (before/after PIN)
- **Dual Authentication** (PIN + Card/Fingerprint/Face)
- **Forced Lock function** (Internal/External)
- **Emergency mechanical key** (models with key)
- **Low battery indicator** (Battery Replacement LED)
- **IR Sensor connection** (Face recognition)
- **Home Network / Remote Control** (optional accessories)
:::

# PIN Registration

## 1. Main PIN Registration (LED [1])

The main PIN opens the door and serves as the password for entering registration/deletion mode.

**Steps:**
1. Open the battery cover on the inner body
2. Press the **Registration (R)** button
3. Enter the **initial PIN `1234`** then press `[#]`
4. LED [1] lights on the keypad
5. Enter the **new PIN** (4-12 digits)
6. Press **R** → melody plays to confirm

:::caution Before you start
- Register while the door is open
- Default PIN is `1, 2, 3, 4` (must change on first use)
- **Only 1 PIN can be registered** — new registration deletes the old one
:::

## 2. Guest PIN Registration (LED [4])

Guest PIN is valid for **1 time only** — perfect for visitors

**Steps:**
1. Press **R** and enter main PIN + `[#]`
2. Press number **[4]** — LED [4] lights
3. Enter the **new Guest PIN** (4-12 digits)
4. Press **R** → melody plays to confirm

:::caution
- **Guest PIN is valid for 1 use only** — auto-deleted after use
- **Guest PIN does NOT require 2nd authentication** — opens door directly
- **Registering a new Guest PIN deletes the old one**
:::

## 3. Super Admin PIN Registration (R button)

Super Admin PIN is used for system reset and advanced management.

**Steps:**
1. Press and hold **OPEN/CLOSE** for **10 seconds**
2. Enter the **current main PIN** then press `[#]`
3. Press the **R** (Registration) button
4. Enter the **new Super Admin PIN**
5. Press **R** again to confirm
6. Melody plays to confirm

## 📋 Cautions

:::warning Possible errors
- **Cannot register PIN that includes Random Numbers** in Registration mode
- **Cannot re-register an existing PIN**
- Must register the **Super Admin PIN** before other registrations
- If the **R button is pressed during a step**, registration is cancelled
:::

## 💡 Tips

- Use a **4-digit PIN** for convenience, or **8-12 digits** for better security
- Don't use guessable PINs like `1234`, `0000`, birth years, or phone numbers
- Change your PIN every 3-6 months
- Store a backup PIN in a safe place (not on the lock)

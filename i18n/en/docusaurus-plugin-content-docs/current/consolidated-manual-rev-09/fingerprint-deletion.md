# Fingerprint Deletion

## 🗑 Delete all (LED [3] — All at once)

:::warning
This deletes all 100 fingerprints at once.
:::

**Steps:**
1. Press **OPEN/CLOSE** for 3 seconds → enter PIN + `[#]`
2. LEDs [2] [3] [5] [0] light up → press **[3]** (Fingerprint)
3. Press and hold **`[#]` for 5 seconds** → melody plays → all fingerprints deleted

## 🗑 Delete by slot (LED [3] — One-by-one)

**Steps:**
1. Press **OPEN/CLOSE** for 3 seconds → enter PIN + `[#]`
2. LEDs [2] [3] [5] [0] light up → press **[3]**
3. Press `[#]` → all keypad numbers light up (standby mode)
4. Enter the **3-digit slot number** (001-100) of the fingerprint to delete
5. Press `[#]` → melody plays → fingerprint deleted
6. Repeat for other fingerprints, or press **R** to finish

## ⚠️ Cautions

:::warning
- **Fingerprint deletion cannot be undone** — must re-register if needed
- **Deleting fingerprints does NOT delete the main PIN** — other settings remain
- If no fingerprint to delete, press **R** to exit
- If an error occurs, LED [0] lights
- **5 consecutive failed scans** = Anti-Prank + 1-minute lockout
- Must register a **valid fingerprint** before deletion
- **All fingerprints will be deleted** if using LED [3] in All at once mode
:::

## 🎯 Registration Mode (User)

Press the R (Registration) button on the inner body. The keypad shows LEDs 1-6 and R:

| LED | Mode | Steps |
|---|---|---|
| **LED [1]** | PIN Number | Enter initial PIN `1234` + `[#]`, enter new PIN → `[R]` |
| **LED [2]** | Card (one-by-one) | Enter PIN + `[#]`, place card → `[R]` |
| **LED [3]** | Fingerprint (one-by-one) | Enter PIN + `[#]`, place finger 4 times → `[R]` |
| **LED [4]** | Guest PIN Number | Enter PIN + `[#]`, enter Guest PIN → `[R]` |
| **LED [5]** | Remote Control (one-by-one) | Enter PIN + `[#]`, press OPEN on remote → `[R]` |
| **LED [6]** | Face ID (one-by-one) | Enter PIN + `[#]`, stare into camera → `[R]` |
| **R Button** | Super Admin PIN | Press `OPEN/CLOSE` + press R for 10 sec → enter new Super Admin PIN |

## 🗑 Deletion Mode (User)

Press `OPEN/CLOSE` for 3 seconds, enter PIN + `[#]`, then select LED:

| LED | Mode | Steps |
|---|---|---|
| **LED [2]** | Card | All: `LED [#]` for 5 sec / One-by-one: enter slot + `LED [#]` |
| **LED [3]** | Fingerprint | All: `LED [#]` for 5 sec / One-by-one: enter slot + `LED [#]` |
| **LED [5]** | Remote Control | All: `LED [#]` for 5 sec / One-by-one: enter slot + `LED [#]` |
| **LED [6]** | Face ID | All: `LED [#]` for 5 sec / One-by-one: enter slot + `LED [#]` |
| **R Button** | User Registered & Settings Reset | Press `OPEN/CLOSE` + press R for 10 sec → delete all |

## 📋 Important notes

- **Only 1 PIN can be registered**
- **Default PIN:** `[1, 2, 3, 4]`
- **Registering a new Guest PIN deletes the old one**
- **Up to 200 cards** can be registered
- **Up to 100 fingerprints** can be registered
- **Up to 30 faces** can be registered
- **Additional authentication (Card/Fingerprint/Remote/Face) does NOT delete the primary authentication (PIN)**
- If primary authentication (PIN) is not registered, dual authentication cannot be set
- If an error occurs during registration, LED [2] will blink
- If the Remote Control receiver module is not connected, LED [5] won't light up → cannot register/delete remotes

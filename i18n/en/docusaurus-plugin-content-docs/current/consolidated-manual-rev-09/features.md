# Special Features (Convenient Features)

## 🔢 Random Number Feature (LED [9])

Add random digits before/after PIN to prevent shoulder-surfing

**Rules:**
- Up to **20 digits** total (including PIN)
- **Must enter PIN after** the Random Number
- Example: `2580` + `1234` + `*` (enter 2580 before 1234)

**Benefits:**
- Prevents onlookers from seeing your PIN
- Clears fingerprint residue on the keypad
- Works for both PIN and Card

## 🔒 Automatic Re-locking Function

**Behavior:**
- If the door is not opened within **7 seconds** after authentication → auto-locks
- Works with PIN, Card, Fingerprint
- No configuration needed — enabled by default

**Example:**
- Open with PIN → don't open door within 7 sec → auto-locks
- Press OPEN/CLOSE → door closes and locks immediately

## ⏰ 1-Minute Lock Warning Function

:::warning
**5 consecutive failed** PIN/Card/Fingerprint attempts → **1-minute lockout**
:::

**Behavior:**
- **After 5 failed attempts:**
  - Alarm sounds for **7 seconds**
  - Lock stops operating for **1 minute**
  - During lockout → Card/PIN/Remote Control won't work
- **Cancel:** Press **OPEN/CLOSE** button
- **After 1 minute:** Lock returns to normal operation

## 🔋 Low Battery Warning Function

**When battery is low:**
- **[Battery Replacement]** LED lights up
- "beep-ree-reep" sound plays
- Authentication may malfunction

**Thresholds:**
- Low voltage: **4.5V** (Face Recognition Door Lock: 5.0V)
- When voltage returns to **5.5V** (Face: 5.6V) → warning stops

:::caution
- **Replace all batteries at once** — don't mix old + new
- Use AA Alkaline batteries (4 pcs)
- Emergency: 9V Alkaline (not included) or USB C-Type (F7000Kr/F9000Kr/P8800K)
:::

## 🔇 Etiquette Mode (LED [9])

Silent mode for one operation

**Steps:**
1. Touch the keypad
2. Press `[#]`

**Effect:**
- **Next authentication** (PIN, Card, Fingerprint, Face, Guest PIN) is silent
- 1-time only, then sound returns
- **5 consecutive failed authentications** → alarm sounds

## 🌙 Sound Mute (LED [9]) — 1-time

Activate by pressing `[*]` in normal state → sound is muted for the next operation

## ⚠️ Etiquette vs Sound Mute

| Feature | Duration | How to activate |
|---|---|---|
| **Etiquette Mode** | 1 time (next open) | Press `[#]` before |
| **Sound Mute** | 1 time (next) | Press `[*]` before |
| **Voice Mode** | Permanent | Configure via LED |
| **Buzzer Mode** (default) | Permanent | Configure via LED |

## 🪟 Automatic Re-locking from Inside

- **Sub Type (sliding door):** Press OPEN/CLOSE button + turn knob
- **Main Type:** Press OPEN/CLOSE button
- **When Deadbolt Auto Lock is on:** Touch keypad for 2 seconds → auto-lock
- **When Deadbolt Auto Lock is off:** Touch keypad for 2 seconds → manual lock

## 🔓 Unlocking from Inside

| Status | How to unlock |
|---|---|
| **No Deadbolt** | Pull the inner body handle |
| **Deadbolt Auto Lock** | Press OPEN/CLOSE for 2 seconds (auto-unlock) |
| **Deadbolt Manual Lock** | Touch keypad for 2 seconds OR press OPEN/CLOSE to unlock |

# Components

## 🏠 Outer body components

| # | Part | Function |
|---|---|---|
| 1 | **Number Pad** | Enter PIN |
| 2 | **Low Battery Lamp** | Lights when battery is low |
| 3 | **Reset Button** | Reset lock to factory defaults |
| 4 | **Smart Card Reader** | Tap registered card here |
| 5 | **Fingerprint Reader** | Place registered finger (supported models) |
| 6 | **Face Recognition Camera** | Look straight into camera (supported models) |
| 7 | **IR Sensor** | Detects proximity |
| 8 | **Emergency Battery Terminal** | Touch 9V battery when dead |
| 9 | **Mechanical Key Hole** | Use emergency key (models with key) |
| 10 | **Outer Body Handle** | Pull down to open door from outside |

## 🔧 Inner body components

| # | Part | Function |
|---|---|---|
| 1 | **Open/Close Button** | Open/close door from inside |
| 2 | **Registration Button (R)** | Enter registration/deletion mode |
| 3 | **Manual Lock Button** | Manual lock |
| 4 | **Smart Card Lamp** | Card status light |
| 5 | **Smart Card Reader (Inner)** | Inner card reader |
| 6 | **Emergency Battery Terminal** | 9V battery slot |
| 7 | **Battery Cover** | Battery compartment cover |
| 8 | **Inner Body Handle** | Pull down to open door from inside |
| 9 | **Battery (AA × 4)** | Main power source |

- Screws
- Strike Cover

### 🔹 MORTISE group — ES-F7000Kr, ES-F9000Kr

- Smart Cards

- Installation Guide

## 🔌 Optional Accessories

| Accessory | Details |
|---|---|
| **Home Network Module** | Wi-Fi connectivity for remote control |
| **Remote Control** | Remote to open door (RX / TRX) |
| **RX, BLE** | Newer models use Bluetooth (ES-P8800K) |
| **Remote Control (BLE)** | Bluetooth remote (ES-P8800K) |

## 📐 Dimensions (examples)

Screw/wire dimensions used:
- **BH (M5×30mm) screws:** 2 pcs
- **BH (M5×20mm) screws:** 2 pcs
- **FHM (M4×8mm) screws:** 4 pcs
- **BH (M4×20mm) screws:** 6 pcs
- **FHM (M5×30mm) screws:** 2 pcs (ES-S520D/H)
- **FHM (M6×12mm) screws:** 4 pcs (ES-S520D/H)

# Components

## Outer body

| # | Part | Function |
|---|---|---|
| 1 | **Number Pad** | Enter PIN or configuration commands |
| 2 | **Battery Lamp** | Lights up when battery is low |
| 3 | **Reset Button** | Reset lock to factory defaults (rarely used) |
| 4 | **Emergency Battery Terminal** | Touch 9V battery here if lock is dead |
| 5 | **RFID Card Reader** | Tap registered card here |

## Inner body

| # | Part | Function |
|---|---|---|
| 1 | **Open/Close Button** | One-touch unlock from inside |
| 2 | **Internal force lock button** |  |
| 3 | **Registration Button** | Used when registering PIN, Card or Fingerprint |
| 4 | **Auto lock sensor** | Detects if the door is closed to auto-lock |
| 5 | **Deadbolt** | Locking bolt that secures the door |
| 6 | **Manual Lock/Unlock Knob** | Turn manually to lock or unlock the door |
| 7 | **Battery Cover** | Slide off to access battery compartment |

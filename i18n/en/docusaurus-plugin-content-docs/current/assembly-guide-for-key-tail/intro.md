# Assembly Guide for Key Tail

This guide explains the step-by-step procedure for assembling the key tail component and spring before installing the **ES-S520 Series** digital door locks.

## 🛠️ Assembly Steps

### Step 1: Prepare Parts
- Take out the Key-tail and spring from the packaging.

### Step 2: Combine Key-tail and Spring
- Combine the Key-tail and spring together.
- **Important:** Ensure the narrow gap between the springs is joined to contact the key tail.

### Step 3: Insert into Key Rod
- Insert the combined spring and Key-tail into the key rod located at the front of the door lock.

### Step 4: Align and Push
- Push and combine the key tail so that it matches with the key rod.
- The spring must cover both the key rod and the key tail at the same time.

### Step 5: Check Connection
- Verify that the Key-tail and key rod are successfully and securely combined.

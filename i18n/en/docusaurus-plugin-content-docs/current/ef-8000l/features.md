# Convenient Features

## 1. Automatic/Manual Lock Setting

Set whether the door locks automatically when closed, or only manually.

**Steps (door half-open):**
1. Press **Manual Lock** button when the deadbolt comes out
2. Touch the number pad
3. Enter PIN then press `[*]` **twice**
4. Press **[4]** for Auto Lock, or **[7]** for Manual Lock

:::note
Auto is the default. In Manual mode, the door does not lock automatically — touch the pad or press the Manual Lock button inside.
:::

## 2. Smart Key Auto/Manual Detection

**Steps:**
1. Touch the number pad
2. Enter PIN then press `[*]` **twice**
3. Press **[1]** for Auto, or **[3]** for Manual

:::note
- **Auto Mode**: place Smart Key on the sensor directly
- **Manual Mode**: touch the pad first, then place Smart Key
- **Auto is the default**
:::

## 3. Sound Mute (1-time)

Mute the operation sound for one use.

**Steps:**
1. Touch the pad, then press `[*]` (in mute state)
2. Authenticate with PIN or Fingerprint

:::note
1-time only. Sound returns on the next operation.
:::

## 4. Sound Volume Setting (7 levels)

**Steps:**
1. Touch the number pad
2. Enter registered PIN then press `[*]`
3. Press **[3]** to increase, or **[6]** to decrease

:::note
- 7 levels maximum
- In mute mode, the pad blinks when you press a number
- Default: high
:::

## 5. Multi-touch Security

Adds random digits to your PIN to clear fingerprint residue and prevent PIN exposure.

**Steps:**
1. Touch the number pad
2. Enter PIN then press `[*]`
3. Press **[5]** — melody plays to confirm
4. Repeat steps 1-3 to disable

:::caution When Multi-touch is enabled
After entering your PIN, the keypad shows 3 random digits — press those digits before `[*]`.
- When Multi-touch is ON → Random Number feature is OFF
- When Random Number is ON → Multi-touch feature is OFF
:::

## 6. Dual-Mode Security (2nd authentication required)

After entering your PIN, you must also authenticate with Smart Key or Fingerprint.

**Steps (requires PIN + Smart Key/Fingerprint already registered):**
1. Press **Register** for **5 seconds** — all LEDs light, after 3 sec alarm stops, after 2 sec lights up again
2. Enter current PIN, then press `[*]` — front panel LED blinks
3. Press **[1]** to turn OFF, or **[3]** to turn ON
4. When confirmation sound plays, done

**How to open the door when Dual-Mode is ON:**
- **PIN + Fingerprint**, or
- **PIN + Smart Card**

:::note Important
- When in Dual-Mode Security Entry, other buttons are disabled
- Must complete 2nd authentication within **10 seconds** or it times out
- Guest PIN does NOT require 2nd authentication — enters directly
- If Guest PIN = User PIN → no 2nd authentication needed
- 2nd authentication follows the same rules as Successful/Failed Authentication and 1-minute Lock
:::

## 7. Home Network Module & Remote Control (Optional)

Open the door remotely via Home Network devices or Remote Control (sold separately).

**Registration:**
1. Press **Registration** — beep
2. Enter PIN then press `[*]`
3. Check that [1, 2, 3, 4, 8, 9] are lit on the pad
4. For **Home Network Module** press **Registration** button / For **Remote Control** press **Open** button
5. Press `[*]` — melody plays, done

:::note
- Register additional remotes by repeating step 4
- **Up to 5 remotes**
:::

**Deletion:**
1. Press **Registration** for 3 seconds
2. Enter PIN then press `[*]`
3. Check that [1, 2, 3, 4, 8, 9] are lit
4. Press **[0]**
5. Press `[*]` and hold for **5 seconds** — melody plays, done

:::note
Even if [0] doesn't light in step 3, you can still press it. New remote registration deletes previous ones — this is an alternative deletion method.
:::

# Components

## Outer body

| # | Part | Function |
|---|---|---|
| 1 | **Number Pad** | Enter PIN or configuration commands |
| 2 | **Battery Lamp** | Lights up when battery is low |
| 3 | **Reset Button** | Reset lock to factory defaults (rarely used) |
| 4 | **RFID Card Reader** | Tap registered card here |
| 5 | **Emergency Battery Terminal** | Touch 9V battery here if lock is dead |

## Inner body

| # | Part | Function |
|---|---|---|
| 1 | **Open/Close Button** | One-touch unlock from inside |
| 2 | **Safety Button** | Press to unlock/lock safely from inside |
| 3 | **Manual Lock/Unlock Knob** | Turn manually to lock or unlock the door |
| 4 | **Registration Button** | Used when registering PIN, Card or Fingerprint |
| 5 | **Deadbolt** | Locking bolt that secures the door |
| 6 | **Battery Cover** | Slide off to access battery compartment |
| 7 | **Alkaline AA Battery** | Main power source |
| 8 | **Battery Cover Fixing Screw** | Secures the battery cover in place |

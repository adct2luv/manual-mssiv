# Components

## Outer body

| # | Part | Function |
|---|---|---|
| 1 | **Fingerprint Reader** | Scan registered fingerprint here |
| 2 | **Number Pad** | Enter PIN or configuration commands |
| 3 | **RFID Card Reader** | Tap registered card here |
| 4 | **Reset Button** | Reset lock to factory defaults (rarely used) |
| 5 | **Emergency Battery Terminal** | Touch 9V battery here if lock is dead |

## Inner body

| # | Part | Function |
|---|---|---|
| 1 | **Battery Cover** | Slide off to access battery compartment |
| 2 | **Registration Button** | Used when registering PIN, Card or Fingerprint |
| 3 | **Manual Lock Button** | Used to close/lock the door in manual mode |

## In the box
- Striker
- Mortise
- 4 Batteries
- Screws (×4)
- Mortise shaft
- Installation guide
- RFID Cards

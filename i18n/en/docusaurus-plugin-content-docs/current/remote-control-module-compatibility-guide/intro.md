# Remote Control Module Compatibility Guide

This guide contains the compatibility matrix and hardware version identification for EPIC remote control transmitters (TX) and receiver modules (RX) across different door lock models.

## 📊 Compatibility Reference Table

| Transmitter (TX) | Receiver (RX) | Compatible EPIC Door Lock Models |
|---|---|---|
| **TX200 & TX300** | **RX200** | EF-8000L, ES-809L, ES-K70, ES-M50 |
| **TX200 & TX300** | **RX300** | TRIPLEX 2WAY, ES-T153, ES-B10 |
| **TX200 & TX300** | **RX500** | ES-303G |
| **TX200 & TX300** | **RX700** | ES-P9100FK |
| **TX600, TRX100, TRX200** | **RX400** | ES-P8800K, ES-9000K, ES-F9000K, ES-F9000Kr, ES-7000K, ES-F7000K, ES-F7000Kr |
| **TX600, TRX100, TRX200** | **RX600 & RX650** | ES-100D, ES-S100D, ES-S100Dr, ES-300D, ES-F300D, ES-F300Dr, ES-F301D, ES-500D, ES-500H, ES-F500D, ES-F500H, ES-F500Dr, ES-F500Hr, ES-F501D, ES-F501H, ES-700G, ES-F700G, ES-FF730G, ES-FF730Gr, ES-FF731G, ES-S520D, ES-S520H |

---

## 🔍 Hardware Identification Codes

To verify module compatibility before purchase, check the printed code on the hardware circuit board:
- **RX600:** `NOR-100-RX 2K14-0323(v1.0)`
- **RX650:** `NOR-RX110-A 2K21-0430(v2.0)`
- **TRX100(19):** `EPR-TRX01_2K19-0705(v1.2)`
- **TRX100(22):** `EPR-TRX100-A_2K22-0710(v0.1)`
- **TRX100(24):** `EPR-TRX100-AU_2K24-0130(V1.0)`

---

## ⚠️ Important Remarks

:::caution[Incompatibility Details]
- The **TRX100(19)** transmitter module **does not work** with the **RX650** receiver module.
- The **TRX100(19)** transmitter module **does not work** with the **TRX200(22)** or **TRX100(22)** modules.
:::
